## Overview

The Command Palette provides a keyboard-driven interface similar to VS Code's Command Palette (Cmd+P) or Raycast. It combines search functionality with keyboard navigation for quick access to actions and commands.

## API Reference

### Command

Main container component with dialog behavior and keyboard shortcuts.

| Prop               | Type                               | Default                         | Description                                     |
| ------------------ | ---------------------------------- | ------------------------------- | ----------------------------------------------- |
| `items`            | `Record<string, CommandItem[]>`    | `{}`                            | Categorized command items (see structure below) |
| `placeholder`      | `string`                           | `'Type a command or search...'` | Input placeholder text                          |
| `emptyText`        | `string`                           | `'No results found.'`           | Text when no results match                      |
| `onSelect`         | `string`                           | `'console.log(item)'`           | Alpine.js expression for selection              |
| `keyboardShortcut` | `string`                           | `'cmd+k'`                       | Global keyboard shortcut                        |
| `variant`          | `'default' \, 'ghost' \, 'dialog'` | `'default'`                     | Visual variant                                  |
| `size`             | `'sm' \, 'md' \, 'lg' \, 'full'`   | `'md'`                          | Size variant                                    |

### CommandInput

| Prop          | Type      | Default                         | Description       |
| ------------- | --------- | ------------------------------- | ----------------- |
| `placeholder` | `string`  | `'Type a command or search...'` | Input placeholder |
| `showIcon`    | `boolean` | `true`                          | Show search icon  |

### CommandItem

Individual command item component.

| Prop        | Type     | Default | Description                  |
| ----------- | -------- | ------- | ---------------------------- |
| `shortcut`  | `string` | -       | Keyboard shortcut to display |
| `className` | `string` | -       | Additional CSS classes       |

### CommandItems

Container for dynamic command lists. Must include `x-for` and `x-key` directives.

### CommandFooter

| Prop        | Type      | Default | Description                    |
| ----------- | --------- | ------- | ------------------------------ |
| `showHints` | `boolean` | `true`  | Show keyboard navigation hints |
| `class`     | `string`  | -       | Additional CSS classes         |

### CommandTrigger

Button or trigger element to open the command palette. Re-exported from DialogTrigger.

| Prop        | Type                                            | Default     | Description            |
| ----------- | ----------------------------------------------- | ----------- | ---------------------- |
| `variant`   | `'default' \, 'button' \, 'outline' \, 'ghost'` | `'default'` | Visual variant         |
| `className` | `string`                                        | -           | Additional CSS classes |

### CommandGroup

Container for grouping command items with an optional title. Re-exported from MenuGroup.

| Prop        | Type     | Default | Description            |
| ----------- | -------- | ------- | ---------------------- |
| `className` | `string` | -       | Additional CSS classes |
| `group`     | `any`    | -       | Dynamic group data     |

### CommandGroups

Dynamic container for multiple command groups. Must include `x-for` and `x-key` directives.

| Prop          | Type      | Default | Description                 |
| ------------- | --------- | ------- | --------------------------- |
| `showDivider` | `boolean` | `false` | Show divider between groups |

### CommandShortcut

Component to display keyboard shortcuts. Re-exported from MenuShortcut.

| Prop        | Type     | Default | Description            |
| ----------- | -------- | ------- | ---------------------- |
| `className` | `string` | -       | Additional CSS classes |
