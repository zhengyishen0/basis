## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| **Tooltip** | | | |
| `delay` | `number` | `500` | Delay in milliseconds before tooltip appears |
| `className` | `string` | - | Additional CSS classes |
| **PopupTrigger** | | | |
| `variant` | `'default' \, 'button' \, 'ghost' \, 'link'` | `'default'` | Visual style of the trigger |
| **PopupContent** | | | |
| `theme` | `'default' \, 'dark' \, 'minimal'` | `'default'` | Visual theme of the tooltip |
| `placement` | `'top' \, 'bottom' \, 'left' \, 'right'` | `'bottom'` | Position relative to trigger |
| `className` | `string` | - | Additional CSS classes |
