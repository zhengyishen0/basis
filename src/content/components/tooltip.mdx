## API Reference

### Tooltip Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `delay` | `number` | `500` | Delay in milliseconds before tooltip appears |
| `className` | `string` | - | Additional CSS classes |

### PopupTrigger Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'button' \| 'ghost' \| 'link'` | `'default'` | Visual style of the trigger |
| `className` | `string` | - | Additional CSS classes |

### PopupContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `theme` | `'default' \| 'dark' \| 'minimal'` | `'default'` | Visual theme of the tooltip |
| `placement` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'bottom'` | Position relative to trigger |
| `className` | `string` | - | Additional CSS classes |
| `role` | `string` | `'dialog'` | ARIA role for accessibility |

### Slots

| Component | Slot | Description |
|-----------|------|-------------|
| PopupTrigger | `default` | Content that triggers the tooltip |
| PopupContent | `default` | Tooltip content |

### Alpine.js Integration

The tooltip automatically manages state using Alpine.js:

```javascript
// Automatically available in x-data scope:
{
  open: false,           // Tooltip visibility state
  trigger: 'hover',      // Trigger type (always 'hover' for tooltips)
  popupId: 'popup-xxx',  // Unique ID for accessibility
  toggle() { ... },      // Toggle tooltip visibility
  close() { ... }        // Close tooltip
}
```

### Usage Patterns

#### Simple Text Tooltip
For basic informational tooltips:

```astro
<Tooltip>
    <PopupTrigger>
        <Button>Action</Button>
    </PopupTrigger>
    <PopupContent placement="top">
        Simple tooltip text
    </PopupContent>
</Tooltip>
```

#### Help Text Tooltip
For help information with custom styling:

```astro
<Tooltip>
    <PopupTrigger>
        <span class="text-blue-600 underline cursor-help">
            Need help?
        </span>
    </PopupTrigger>
    <PopupContent placement="top" class="max-w-xs">
        <div class="text-sm">
            Detailed help information goes here.
        </div>
    </PopupContent>
</Tooltip>
```

#### Rich Content Tooltip
For tooltips with complex content:

```astro
<Tooltip>
    <PopupTrigger>
        <Button variant="outline">Feature Info</Button>
    </PopupTrigger>
    <PopupContent placement="right" class="max-w-sm">
        <div class="space-y-2">
            <div class="font-semibold">Feature Name</div>
            <div class="text-sm text-muted-foreground">
                This feature provides advanced functionality...
            </div>
            <div class="flex gap-2 mt-2">
                <span class="text-xs bg-blue-100 text-blue-700 px-2 py-1 rounded">
                    Pro
                </span>
            </div>
        </div>
    </PopupContent>
</Tooltip>
```

#### Custom Delay Tooltip
For tooltips that need different timing:

```astro
<!-- Quick tooltip for frequent interactions -->
<Tooltip delay={100}>
    <PopupTrigger>
        <Button size="sm">Quick Action</Button>
    </PopupTrigger>
    <PopupContent>Quick info</PopupContent>
</Tooltip>

<!-- Delayed tooltip to reduce noise -->
<Tooltip delay={1000}>
    <PopupTrigger>
        <span class="cursor-help">Complex Feature</span>
    </PopupTrigger>
    <PopupContent>Detailed explanation after delay</PopupContent>
</Tooltip>
```

### Accessibility

The tooltip component includes built-in accessibility features:

- ARIA attributes for screen readers
- Keyboard navigation support (Escape to close)
- Focus management
- Semantic HTML structure

### Styling Notes

- Tooltips inherit the site's design system colors
- Use `className` prop for custom styling
- Content automatically wraps with `max-w-xs` by default
- Dark theme provides high contrast for better readability
- Minimal theme offers subtle styling for clean interfaces
