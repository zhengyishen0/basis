## Overview

The Divider component is a simple utility component for creating visual separation between content sections. It supports both horizontal and vertical orientations with customizable thickness, color, and spacing options.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `direction` | `'horizontal' | 'vertical'` | `'horizontal'` | Divider orientation |
| `color` | `'gray' | 'neutral' | 'slate'` | `'gray'` | Divider color |
| `thickness` | `'1' | '2' | '4'` | `'1'` | Divider thickness in pixels |
| `spacing` | `'2' | '4' | '6' | '8'` | `'4'` | Spacing around divider |
| `class` | `string` | - | Additional CSS classes |
| `...rest` | `any` | - | Other HTML attributes |

### Direction Options

- **`horizontal`** (default): Creates a horizontal line divider with width and margin-y
- **`vertical`**: Creates a vertical line divider with height and margin-x

### Color Options

- **`gray`** (default): `bg-gray-200`
- **`neutral`**: `bg-neutral-200`  
- **`slate`**: `bg-slate-200`

### Thickness Options

- **`1`** (default): 1px thickness (`h-px` for horizontal, `w-px` for vertical)
- **`2`**: 2px thickness (`h-0.5` for horizontal, `w-0.5` for vertical)
- **`4`**: 4px thickness (`h-1` for horizontal, `w-1` for vertical)

### Spacing Options

- **`2`**: `my-2` (horizontal) or `mx-2` (vertical)
- **`4`** (default): `my-4` (horizontal) or `mx-4` (vertical)
- **`6`**: `my-6` (horizontal) or `mx-6` (vertical)
- **`8`**: `my-8` (horizontal) or `mx-8` (vertical)

### Accessibility

- Includes `role="separator"` for screen readers
- Includes `aria-orientation` attribute matching the direction

## Design Principles

Divider is a **simple utility component** that:
- Uses `cn()` utility with conditionals for dynamic styling
- Provides both horizontal and vertical orientations
- Supports customizable thickness, color, and spacing
- Includes proper accessibility attributes
- Can be easily customized with additional CSS classes
- Perfect for creating visual separation in layouts, cards, and menus

## Common Use Cases

- **Card Sections**: Separate different areas within cards
- **Menu Items**: Divide navigation or dropdown menu items
- **Content Sections**: Create visual breaks between content areas
- **Toolbar Separation**: Separate groups of buttons or tools
- **Form Sections**: Divide form fields into logical groups
