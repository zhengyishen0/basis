## Overview

A simple utility component for creating visual separation between content sections. Supports both horizontal and vertical orientations with customizable styling.

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `direction` | `'horizontal' \| 'vertical'` | `'horizontal'` | Divider orientation |
| `color` | `'gray' \| 'neutral' \| 'slate'` | `'gray'` | Divider color |
| `thickness` | `'1' \| '2' \| '4'` | `'1'` | Divider thickness in pixels |
| `spacing` | `'0' \| '2' \| '4' \| '6' \| '8'` | `'4'` | Spacing around divider |
| `class` | `string` | - | Additional CSS classes |
