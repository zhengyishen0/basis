## Overview

Row is a simple layout component that arranges elements horizontally using flexbox.

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `overflow` | `'auto' \, 'fixed' \, 'expand' \, 'wrap' \, 'show'` | `'expand'` | Overflow behavior |
| `class` | `string` | - | Additional CSS classes |
| `...alpineProps` | `any` | - | Alpine.js attributes |
