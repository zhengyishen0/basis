## Overview

The HoverCard component displays rich content when hovering over elements. Perfect for user profiles, product previews, or detailed information cards.

## API Reference

| Component            | Prop        | Type                                                       | Default    | Description                          |
| -------------------- | ----------- | ---------------------------------------------------------- | ---------- | ------------------------------------ |
| **HoverCard**        | `delay`     | `number`                                                   | `200`      | Delay before hover card appears (ms) |
|                      | `className` | `string`                                                   | -          | Additional CSS classes               |
| **HoverCardTrigger** | `className` | `string`                                                   | -          | Additional CSS classes               |
| **HoverCardContent** | `placement` | `'top' \| 'bottom' \| 'left' \| 'right' \| 'bottom-start'` | `'bottom'` | Position relative to trigger         |
|                      | `size`      | `'sm' \| 'md' \| 'lg'`                                     | `'md'`     | Size of the hover card               |
|                      | `className` | `string`                                                   | -          | Additional CSS classes               |
