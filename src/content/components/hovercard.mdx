## API Reference

| Component | Prop | Type | Default | Description |
|-----------|------|------|---------|-------------|
| **HoverCard** | `delay` | `number` | `200` | Delay before hover card appears (ms) |
| | `className` | `string` | - | Additional CSS classes |
| **PopupTrigger** | `variant` | `'default' \| 'button' \| 'ghost' \| 'link'` | `'default'` | Visual style of trigger |
| | `className` | `string` | - | Additional CSS classes |
| **PopupContent** | `placement` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'bottom'` | Position relative to trigger |
| | `theme` | `'default' \| 'dark' \| 'minimal'` | `'default'` | Visual theme |
| | `className` | `string` | - | Additional CSS classes |
