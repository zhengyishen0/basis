## Overview

The Textarea component provides a multi-line text input field with support for auto-resize functionality, universal theme styling, and full Alpine.js integration for reactive behavior.

## API Reference

| Prop          | Type      | Default | Description                        |
| ------------- | --------- | ------- | ---------------------------------- |
| `placeholder` | `string`  | -       | Placeholder text                   |
| `value`       | `string`  | -       | Initial value                      |
| `disabled`    | `boolean` | `false` | Whether the textarea is disabled   |
| `readonly`    | `boolean` | `false` | Whether the textarea is read-only  |
| `required`    | `boolean` | `false` | Whether the field is required      |
| `name`        | `string`  | -       | Name attribute for form submission |
| `id`          | `string`  | -       | ID attribute for labeling          |
| `rows`        | `number`  | `4`     | Number of visible text lines       |
| `autoResize`  | `boolean` | `false` | Enable automatic height adjustment |
| `className`   | `string`  | -       | Additional CSS classes             |
