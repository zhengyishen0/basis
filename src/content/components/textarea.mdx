## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | - | Placeholder text |
| `value` | `string` | - | Initial value |
| `disabled` | `boolean` | `false` | Whether the textarea is disabled |
| `readonly` | `boolean` | `false` | Whether the textarea is read-only |
| `required` | `boolean` | `false` | Whether the field is required |
| `name` | `string` | - | Name attribute for form submission |
| `id` | `string` | - | ID attribute for labeling |
| `rows` | `number` | `4` | Number of visible text lines |
| `autoResize` | `boolean` | `false` | Enable automatic height adjustment |
| `className` | `string` | - | Additional CSS classes |

### Auto-resize Feature

When `autoResize` is enabled, the textarea automatically adjusts its height based on content:

- Dynamically grows as content is added
- Shrinks when content is removed
- Maintains minimum height based on `rows` prop
- Uses Alpine.js for smooth, reactive resizing
- No maximum height by default (can be controlled with CSS)

### Alpine.js Integration

The component supports all Alpine.js directives:

- Use `x-model` for two-way data binding
- Access reactive properties and methods
- Integrate with Alpine.js state management
- Auto-resize functionality uses Alpine.js internally when enabled

### Accessibility Features

- Proper semantic HTML with native textarea element
- Full keyboard navigation support
- Screen reader compatibility
- Supports ARIA attributes via pass-through props
- Focus management with visible focus indicators

## Usage Examples

### Character Counter

```astro
<div x-data="{ 
    content: '', 
    maxLength: 280,
    get remainingChars() {
        return this.maxLength - this.content.length;
    },
    get isOverLimit() {
        return this.content.length > this.maxLength;
    }
}">
    <div class="space-y-2">
        <label class="block text-sm font-medium">Tweet (280 characters max)</label>
        <Textarea 
            x-model="content"
            placeholder="What's happening?"
            rows={3}
            autoResize
            maxlength="300"
        />
        <div class="flex justify-between items-center text-sm">
            <span 
                x-text="content.length + ' / ' + maxLength + ' characters'"
                x-bind:class="isOverLimit ? 'text-destructive' : 'text-muted-foreground'"
            ></span>
            <button 
                type="button"
                x-bind:disabled="content.length === 0 || isOverLimit"
                class="px-3 py-1 bg-primary text-primary-foreground rounded disabled:opacity-50"
            >
                Tweet
            </button>
        </div>
    </div>
</div>
```

### Template System

```astro
<div x-data="{ 
    selectedTemplate: '',
    content: '',
    templates: {
        email: 'Subject: \\n\\nDear [Name],\\n\\n\\n\\nBest regards,\\n[Your Name]',
        meeting: 'Meeting: [Topic]\\nDate: [Date]\\nTime: [Time]\\n\\nAgenda:\\n1. \\n2. \\n3. ',
        report: '# Report Title\\n\\n## Summary\\n\\n## Key Findings\\n\\n## Recommendations\\n\\n## Conclusion'
    },
    loadTemplate(template) {
        this.content = this.templates[template] || '';
    }
}">
    <div class="space-y-4">
        <div class="flex gap-2 flex-wrap">
            <button 
                x-on:click="loadTemplate('email')"
                class="px-3 py-1 text-sm bg-secondary text-secondary-foreground rounded hover:bg-secondary/80"
            >
                Email Template
            </button>
            <button 
                x-on:click="loadTemplate('meeting')"
                class="px-3 py-1 text-sm bg-secondary text-secondary-foreground rounded hover:bg-secondary/80"
            >
                Meeting Notes
            </button>
            <button 
                x-on:click="loadTemplate('report')"
                class="px-3 py-1 text-sm bg-secondary text-secondary-foreground rounded hover:bg-secondary/80"
            >
                Report Template
            </button>
            <button 
                x-on:click="content = ''"
                class="px-3 py-1 text-sm bg-muted text-muted-foreground rounded hover:bg-muted/80"
            >
                Clear
            </button>
        </div>
        
        <Textarea 
            x-model="content"
            placeholder="Select a template above or start typing..."
            rows={12}
            autoResize
            class="font-mono text-sm"
        />
        
        <div class="text-xs text-muted-foreground">
            Lines: <span x-text="content.split('\\n').length"></span> | 
            Words: <span x-text="content.trim() ? content.trim().split(/\\s+/).length : 0"></span> |
            Characters: <span x-text="content.length"></span>
        </div>
    </div>
</div>
```
