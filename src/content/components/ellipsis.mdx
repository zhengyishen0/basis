## Use Cases

The Ellipsis component is commonly used for:

- **Table Actions**: Row-level actions in data tables
- **Card Menus**: Overflow actions in card components
- **List Items**: Additional options for list entries
- **Navigation**: More options in navigation menus
- **Mobile Interfaces**: Space-saving action triggers
- **Context Menus**: Visual trigger for contextual actions

## Design Guidelines

### Placement
- Position at the end of content rows or in corners of cards
- Maintain consistent alignment across similar components
- Ensure adequate spacing from other interactive elements

### Interaction
- Provide clear hover states to indicate interactivity
- Use with dropdown menus, popovers, or modal dialogs
- Include keyboard navigation support for accessibility

### Visual Hierarchy
- Keep ellipsis styling subtle but discoverable
- Use consistent sizing within the same interface section
- Consider color variations for different action types

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | - | Additional CSS classes for customization |

### Features

- **Accessible**: Includes proper ARIA attributes and role
- **Interactive**: Built-in hover and focus states
- **Customizable**: Accepts custom classes for styling
- **Alpine.js Compatible**: Works seamlessly with Alpine.js events
- **Responsive**: Maintains proper sizing across devices

### Default Styling

The component includes these default styles:
- `h-9 w-9` - Standard square dimensions
- `rounded-md` - Subtle border radius
- `hover:bg-muted` - Hover background color
- `transition-colors` - Smooth color transitions
- `cursor-pointer` - Pointer cursor on hover

### Integration Examples

#### With Dropdown Menu
```astro
<div x-data="{ open: false }">
    <Ellipsis x-on:click="open = !open" />
    <div x-show="open" x-on:click.away="open = false">
        <!-- Menu items -->
    </div>
</div>
```

#### With Tooltip
```astro
<div x-data="{ tooltip: false }">
    <Ellipsis 
        x-on:mouseenter="tooltip = true" 
        x-on:mouseleave="tooltip = false"
    />
    <div x-show="tooltip" class="tooltip">More actions</div>
</div>
```

### Accessibility

- Uses `role="presentation"` and `aria-hidden="true"` for screen readers
- Focusable and keyboard accessible when used with interactive elements
- Maintains proper color contrast ratios
- Works with assistive technologies when properly labeled
