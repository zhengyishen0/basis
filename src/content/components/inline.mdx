## Overview

The Inline component is a content-driven component that arranges elements horizontally with gap spacing. It has a default `max-w-full` constraint and `wrap` overflow behavior. Perfect for tags, badges, button groups, and any content that should flow horizontally and wrap to new lines when needed.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `overflow` | `'auto', 'fixed', 'expand', 'wrap', 'show'` | `'wrap'` | Controls overflow behavior |
| `class` | `string` | - | Additional CSS classes |
| `...alpineProps` | `any` | - | Alpine.js attributes (x-data, x-for, etc.) |

### Base Classes

- **`flex flex-row justify-start items-center max-w-full gap-2`**
- Horizontal layout with left alignment
- Items centered vertically
- Default maximum width constraint
- Small gap between items

### Overflow Behaviors

- **`wrap`** (default): Items wrap to new lines using flex-wrap
- **`auto`**: Horizontal scrolling when content overflows
- **`fixed`**: Content is clipped at container boundaries
- **`expand`**: Container grows naturally (removes max-w-full)
- **`show`**: Content can spill outside container bounds

### Default Constraints

- **`max-w-full`**: Default maximum width constraint (can be overridden)
- **`flex flex-row`**: Horizontal layout with flexbox

### Slots

| Slot | Description |
|------|-------------|
| default | Items to be arranged horizontally with optional wrapping |

## Design Principles

Inline is a **content-driven component** that:
- Only has gap property for spacing between items
- Size determined by internal elements with sensible default constraints
- Handles dynamic/data-driven content
- Default `max-w-full` with `wrap` overflow for natural line wrapping
- Perfect for tags, badges, button groups, and breadcrumbs
- Default wrapping behavior makes it ideal for responsive tag collections
- Can be easily overridden with custom CSS classes for specific width constraints

## Common Use Cases

- **Tag Collections**: Perfect for displaying multiple tags that wrap naturally
- **Button Toolbars**: Horizontal button groups that can wrap on smaller screens
- **Breadcrumb Navigation**: Linear navigation elements with separators
- **Filter Pills**: Interactive filter options that wrap when numerous
- **Social Media Links**: Horizontal social icons that adapt to container width
