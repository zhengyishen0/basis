## Menu Components

### MenuContainer
A sophisticated container that provides search, keyboard navigation, and state management for complex menu systems.

### MenuItem & MenuItems
Individual menu items with support for icons, shortcuts, and dynamic rendering.

### MenuGroup & MenuGroups
Organize menu items into logical groups with optional labels.

### MenuInput
Search input component with keyboard navigation support.

### MenuShortcut
Display keyboard shortcuts in menu items.

### MenuSubMenu & MenuSubTrigger
Create nested submenus with proper positioning and hover behavior.

### MenuConditional
Conditionally render menu items based on state or permissions.

## API Reference

### MenuBar Props

| Prop        | Type     | Default | Description                |
| ----------- | -------- | ------- | -------------------------- |
| `className` | `string` | -       | Additional CSS classes     |

### MenuContainer Props

| Prop             | Type                                | Default       | Description                          |
| ---------------- | ----------------------------------- | ------------- | ------------------------------------ |
| `variant`        | `'default' \| 'ghost' \| 'dialog'`  | `'default'`   | Visual style variant                 |
| `size`           | `'sm' \| 'md' \| 'lg' \| 'full'`    | `'md'`        | Container size                       |
| `items`          | `Record<string, MenuItem[]>`        | `{}`          | Static menu items data               |
| `searchEnabled`  | `boolean`                           | `true`        | Enable search functionality          |
| `keyboardNav`    | `boolean`                           | `true`        | Enable keyboard navigation           |
| `searchModel`    | `string`                            | `'menuSearch'` | Alpine.js model for search           |
| `activeModel`    | `string`                            | `'activeIndex'` | Alpine.js model for active item     |
| `itemsModel`     | `string`                            | `'menuItems'` | Alpine.js model for items            |
| `emptyText`      | `string`                            | `'No results found.'` | Text when no results         |
| `onSelect`       | `string`                            | `'console.log(item)'` | Item selection handler       |

### MenuItem Props

| Prop          | Type                               | Default     | Description                        |
| ------------- | ---------------------------------- | ----------- | ---------------------------------- |
| `variant`     | `'default' \| 'command'`           | `'default'` | Visual style variant               |
| `shortcut`    | `string`                           | -           | Keyboard shortcut to display       |
| `disabled`    | `boolean`                          | `false`     | Whether the item is disabled       |
| `activeModel` | `string`                           | `'activeIndex'` | Alpine.js model for active state   |
| `role`        | `'menuitem' \| 'option' \| 'button'` | `'menuitem'` | ARIA role                         |
| `onSelect`    | `string`                           | -           | Click handler function             |
| `item`        | `any`                              | -           | Dynamic item data                  |

### MenuInput Props

| Prop          | Type      | Default              | Description                     |
| ------------- | --------- | -------------------- | ------------------------------- |
| `placeholder` | `string`  | `'Search...'`        | Input placeholder text          |
| `showIcon`    | `boolean` | `true`               | Show search icon                |
| `searchModel` | `string`  | `'menuSearch'`       | Alpine.js model for search      |
| `onNext`      | `string`  | `'itemActiveNext()'` | Next item handler               |
| `onPrevious`  | `string`  | `'itemActivePrevious()'` | Previous item handler       |
| `onSelect`    | `string`  | `'selectActiveItem()'` | Select handler                |
| `onEscape`    | `string`  | Reset search         | Escape key handler              |

### MenuGroup Props

| Prop        | Type     | Default | Description                |
| ----------- | -------- | ------- | -------------------------- |
| `className` | `string` | -       | Additional CSS classes     |
| `group`     | `any`    | -       | Dynamic group data         |

## Usage Patterns

### Application Menus
Perfect for desktop-style applications with File, Edit, View menus.

### Command Palettes
Ideal for search-driven command interfaces like VS Code's command palette.

### Project/File Browsers
Great for hierarchical navigation with search and filtering.

### Settings Menus
Excellent for organizing complex settings and preferences.

## Keyboard Navigation

All menu components support keyboard navigation:

- **↑/↓**: Navigate between items
- **↵**: Select active item
- **Esc**: Close menu or clear search
- **Type**: Search (when search is enabled)

## Advanced Features

### Search and Filtering
MenuContainer provides built-in search functionality that filters items across all groups.

### State Management
Comprehensive Alpine.js integration for managing active states, selections, and dynamic content.

### Accessibility
Full ARIA support with proper roles, labels, and keyboard navigation.
