## Overview

MenuBar provides application-style menu bars with File, Edit, View menus. Features comprehensive menu components for building complex, searchable, and interactive menu systems with keyboard navigation and dynamic content.

## API Reference

| Component         | Prop            | Type                     | Default       | Description                 |
| ----------------- | --------------- | ------------------------ | ------------- | --------------------------- |
| **MenuBar**       | `trigger`       | `string`                 | `hover`       | Trigger behavior            |
|                   | `className`     | `string`                 | -             | Additional CSS classes      |
| **MenuContainer** | `size`          | `'sm' \, 'md' \, 'lg'`   | `'md'`        | Container size              |
|                   | `searchEnabled` | `boolean`                | `true`        | Enable search functionality |
|                   | `variant`       | `'default' \, 'ghost'`   | `'default'`   | Visual style variant        |
| **MenuItem**      | `disabled`      | `boolean`                | `false`       | Whether item is disabled    |
|                   | `shortcut`      | `string`                 | -             | Keyboard shortcut           |
|                   | `variant`       | `'default' \, 'command'` | `'default'`   | Visual style variant        |
| **MenuInput**     | `placeholder`   | `string`                 | `'Search...'` | Input placeholder           |
|                   | `showIcon`      | `boolean`                | `true`        | Show search icon            |
| **MenuGroup**     | `className`     | `string`                 | -             | Additional CSS classes      |
