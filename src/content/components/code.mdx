## Overview

The Code component provides syntax highlighting for various programming languages with built-in copy-to-clipboard functionality using Shiki.

## Supported Languages

The Code component supports syntax highlighting for numerous programming languages including:

- **Web**: HTML, CSS, JavaScript, TypeScript, JSX, TSX
- **Backend**: Python, Java, C#, Go, Rust, PHP
- **Shell**: Bash, PowerShell, Zsh
- **Config**: JSON, YAML, TOML, XML
- **Markup**: Markdown, MDX
- **Databases**: SQL, PostgreSQL, MySQL

## Themes

Available syntax highlighting themes:

- `github-light` (default)
- `github-dark`
- `vitesse-light`
- `vitesse-dark`

## API Reference

| Prop       | Type                                                                   | Default          | Description                                   |
| ---------- | ---------------------------------------------------------------------- | ---------------- | --------------------------------------------- |
| `code`     | `string`                                                               | -                | **Required.** The code content to highlight   |
| `language` | `string`                                                               | `'javascript'`   | Programming language for syntax highlighting  |
| `title`    | `string`                                                               | -                | Optional title displayed above the code block |
| `showCopy` | `boolean`                                                              | `true`           | Show/hide copy-to-clipboard button            |
| `theme`    | `'github-light' \, 'github-dark' \, 'vitesse-light' \, 'vitesse-dark'` | `'github-light'` | Syntax highlighting theme                     |
| `height`   | `string`                                                               | `'400px'`        | Height of the code container with scrolling   |
| `class`    | `string`                                                               | -                | Additional CSS classes                        |
