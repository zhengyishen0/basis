## Supported Languages

The Code component supports syntax highlighting for numerous programming languages including:

- **Web**: HTML, CSS, JavaScript, TypeScript, JSX, TSX
- **Backend**: Python, Java, C#, Go, Rust, PHP
- **Shell**: Bash, PowerShell, Zsh
- **Config**: JSON, YAML, TOML, XML
- **Markup**: Markdown, MDX
- **Databases**: SQL, PostgreSQL, MySQL

## Themes

Available syntax highlighting themes:

- `github-light` (default)
- `github-dark`
- `vitesse-light`
- `vitesse-dark`

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `code` | `string` | - | **Required.** The code content to highlight |
| `language` | `string` | `'javascript'` | Programming language for syntax highlighting |
| `title` | `string` | - | Optional title displayed above the code block |
| `showCopy` | `boolean` | `true` | Show/hide copy-to-clipboard button |
| `theme` | `'github-light' \, 'github-dark' \, 'vitesse-light' \, 'vitesse-dark'` | `'github-light'` | Syntax highlighting theme |
| `class` | `string` | - | Additional CSS classes |

### Features

- **Syntax Highlighting**: Powered by Shiki for accurate, fast highlighting
- **Copy to Clipboard**: Built-in copy functionality with visual feedback
- **Theme Support**: Multiple color schemes for different environments
- **Language Detection**: Supports 100+ programming languages
- **Responsive Design**: Adapts to container width with horizontal scrolling
- **Alpine.js Compatible**: Supports reactive props and event handling

### Usage Tips

- Use descriptive titles for complex code examples
- Choose themes that match your site's design system
- Disable copy functionality for short snippets or terminal outputs
- Consider using `vitesse-dark` theme for dark mode interfaces
