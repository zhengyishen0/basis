## Overview

The DataTable component is a comprehensive solution for displaying tabular data with built-in features like sorting, filtering, pagination, row selection, and column visibility controls. It's built with Alpine.js for reactive functionality and includes a powerful state management system.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `data` | `any[]` | `[]` | Array of data objects to display |
| `columns` | `Column[]` | `[]` | Column definitions (see Column interface below) |
| `variant` | `'default', 'bordered', 'striped', 'hoverable'` | `'default'` | Table visual variant |
| `density` | `'compact', 'normal', 'comfortable'` | `'normal'` | Row padding density |
| `size` | `'sm', 'md', 'lg'` | `'md'` | Table text size |
| `interactive` | `'none', 'hover', 'click'` | `'none'` | Row interaction behavior |
| `responsive` | `boolean` | `true` | Enable horizontal scrolling on small screens |
| `searchable` | `boolean` | `false` | Enable global search functionality |
| `selectable` | `boolean` | `false` | Enable row selection with checkboxes |
| `paginated` | `boolean` | `false` | Enable pagination |
| `pageSize` | `number` | `10` | Number of rows per page |
| `loading` | `boolean` | `false` | Show loading state |
| `emptyMessage` | `string` | `'No data available'` | Message shown when no data |
| `class` | `string` | - | Additional CSS classes |

### Column Interface

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `key` | `string` | - | Property key from data object |
| `label` | `string` | - | Column header text |
| `sortable` | `boolean` | `false` | Enable sorting for this column |
| `filterable` | `boolean` | `false` | Include column in global search |
| `align` | `'left', 'center', 'right'` | `'left'` | Text alignment |
| `width` | `string` | - | Column width (CSS class) |
| `hidden` | `boolean` | `false` | Initially hide column |
| `render` | `(value: any, row: any) => string` | - | Custom cell renderer |

### Slots

| Slot | Description |
|------|-------------|
| `toolbar` | Custom toolbar content (typically TableToolbar component) |
| `pagination` | Custom pagination content (typically TablePagination component) |

### Alpine.js Store Methods

The DataTable creates an Alpine.js store with these methods:

| Method | Parameters | Description |
|--------|------------|-------------|
| `sort(columnKey)` | `string` | Sort by column key |
| `filter(query?)` | `string?` | Filter data by search query |
| `goToPage(page)` | `number` | Navigate to specific page |
| `nextPage()` | - | Go to next page |
| `previousPage()` | - | Go to previous page |
| `setPageSize(size)` | `number` | Change page size |
| `toggleRowSelection(rowId)` | `any` | Toggle row selection |
| `toggleSelectAll()` | - | Toggle all rows selection |
| `clearSelection()` | - | Clear all selections |
| `toggleColumn(columnKey)` | `string` | Toggle column visibility |
| `updateData(newData)` | `any[]` | Update table data |
| `refreshData()` | - | Emit refresh event |

### Events

| Event | Payload | Description |
|-------|---------|-------------|
| `table-refresh` | - | Emitted when refreshData() is called |

## Usage Examples

### Dynamic Data Loading

```astro
---
// In your Astro component
const initialData = await fetchUsers();
---

<div x-data="{ tableData: [] }" x-init="tableData = await fetch('/api/users').then(r => r.json())">
  <DataTable 
    x-bind:data="tableData"
    columns={columns}
    searchable={true}
    paginated={true}
    x-on:table-refresh="tableData = await fetch('/api/users').then(r => r.json())"
  />
</div>
```

### Custom Cell Rendering

```javascript
const columns = [
  {
    key: 'avatar',
    label: 'Avatar',
    render: (value, row) => `
      <div class="flex items-center gap-2">
        <img src="${value}" alt="${row.name}" class="w-8 h-8 rounded-full" />
        <span>${row.name}</span>
      </div>
    `
  },
  {
    key: 'actions',
    label: 'Actions',
    align: 'center',
    render: (value, row) => `
      <div class="flex gap-1">
        <button onclick="editItem(${row.id})" class="btn-sm btn-primary">Edit</button>
        <button onclick="deleteItem(${row.id})" class="btn-sm btn-destructive">Delete</button>
      </div>
    `
  }
];
```

### Working with Selection

```astro
<div x-data="{ selectedItems: [] }">
  <DataTable 
    data={items}
    columns={columns}
    selectable={true}
    x-on:selection-changed="selectedItems = $event.detail"
  />
  
  <div x-show="selectedItems.length > 0">
    <span x-text="selectedItems.length + ' items selected'"></span>
    <button x-on:click="bulkDelete(selectedItems)">Delete Selected</button>
  </div>
</div>
```
