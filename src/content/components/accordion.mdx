## Overview

A collapsible accordion component built with Alpine.js that supports both static and dynamic content. Features smooth animations with x-collapse directive and customizable icons.

## API Reference

### Accordion

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `maxWidth` | `'sm' \| 'md' \| 'lg' \| 'xl' \| 'full'` | `'md'` | Maximum width |
| `centered` | `boolean` | `true` | Center horizontally |
| `class` | `string` | - | Additional CSS classes |

### AccordionItem

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'elevated' \| 'flat'` | `'default'` | Visual style variant |
| `class` | `string` | - | Additional CSS classes |

### AccordionTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `iconType` | `'chevron' \| 'plus'` | `'chevron'` | Icon type |
| `variant` | `'default' \| 'subtle'` | `'default'` | Visual style |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size |
| `class` | `string` | - | Additional CSS classes |

### AccordionContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'emphasized'` | `'default'` | Visual style |
| `class` | `string` | - | Additional CSS classes |
