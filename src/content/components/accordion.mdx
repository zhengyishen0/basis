## Overview

A collapsible accordion component built with Alpine.js that supports both static and dynamic content rendering. Features smooth animations with Alpine's x-collapse directive and flexible icon options.

## API Reference

### Accordion Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `maxWidth` | `'sm' | 'md' | 'lg' | 'xl' | 'full'` | `'md'` | Maximum width of accordion |
| `centered` | `boolean` | `true` | Center the accordion horizontally |
| `overflow` | `'fixed' | 'auto' | 'hidden' | 'visible'` | `'fixed'` | Overflow handling |
| `class` | `string` | - | Additional CSS classes |

### AccordionItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' | 'elevated' | 'flat'` | `'default'` | Visual style variant |
| `class` | `string` | - | Additional CSS classes |

### AccordionTrigger Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `iconType` | `'chevron' | 'plus'` | `'chevron'` | Icon type for expand/collapse |
| `text` | `string` | - | Text content (alternative to slots) |
| `variant` | `'default' | 'subtle'` | `'default'` | Visual style variant |
| `size` | `'sm' | 'md' | 'lg'` | `'md'` | Size of the trigger |
| `overflow` | `'fixed' | 'auto' | 'hidden' | 'visible'` | `'fixed'` | Overflow handling |
| `class` | `string` | - | Additional CSS classes |

### AccordionContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | - | Text content (alternative to slots) |
| `variant` | `'default' | 'emphasized'` | `'default'` | Visual style variant |
| `overflow` | `'fixed' | 'auto' | 'hidden' | 'visible'` | `'fixed'` | Overflow handling |
| `class` | `string` | - | Additional CSS classes |

### Slots

| Component | Slot | Description |
|-----------|------|-------------|
| AccordionTrigger | `default` | Trigger button content |
| AccordionTrigger | `icon` | Custom icon content |
| AccordionContent | `default` | Accordion panel content |

### Alpine.js Integration

The accordion automatically manages state using Alpine.js:

```javascript
// Automatically available in x-data scope:
{
  activeAccordion: '',           // Currently active accordion ID
  setActiveAccordion(id) { ... } // Function to toggle accordion state
}
```

### Usage Patterns

#### Static Content
Use `AccordionItem` for static, predefined content:

```astro
<Accordion>
    <AccordionItem>
        <AccordionTrigger>Static Question</AccordionTrigger>
        <AccordionContent>Static Answer</AccordionContent>
    </AccordionItem>
</Accordion>
```

#### Dynamic Content  
Use `AccordionItems` with `x-for` for dynamic data:

```astro
<Accordion>
    <AccordionItems x-for="item in items" x-key="item.id">
        <AccordionTrigger>
            <span x-text="item.question"></span>
        </AccordionTrigger>
        <AccordionContent>
            <span x-text="item.answer"></span>
        </AccordionContent>
    </AccordionItems>
</Accordion>
```

#### Custom Icons
Provide custom icons through the icon slot:

```astro
<AccordionTrigger>
    Question Text
    <div slot="icon" class="custom-icon">
        {/* Custom icon content */}
    </div>
</AccordionTrigger>
```
