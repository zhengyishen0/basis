## Overview

DropdownMenu displays a menu of actions or options in a dropdown overlay. It's commonly used for user menus, action buttons, and as an alternative to select components when you need more complex menu items.

## Usage Patterns

### User Menus
Use dropdown menus for user account actions, profile settings, and authentication options.

### Action Menus
Perfect for context-sensitive actions on table rows, cards, or other content items.

### Select Alternative
When you need more complex menu items than a standard select allows (icons, descriptions, shortcuts).

## API Reference

### DropdownMenu Props

| Prop        | Type                                     | Default   | Description                    |
| ----------- | ---------------------------------------- | --------- | ------------------------------ |
| `trigger`   | `'click' \| 'hover' \| 'focus' \| 'manual'` | `'click'` | How the menu is triggered      |
| `className` | `string`                                 | -         | Additional CSS classes         |

### MenuItem Props

| Prop          | Type                               | Default     | Description                        |
| ------------- | ---------------------------------- | ----------- | ---------------------------------- |
| `variant`     | `'default' \| 'command'`           | `'default'` | Visual style variant               |
| `disabled`    | `boolean`                          | `false`     | Whether the item is disabled       |
| `shortcut`    | `string`                           | -           | Keyboard shortcut to display       |
| `activeModel` | `string`                           | -           | Alpine.js model for active state   |
| `onSelect`    | `string`                           | -           | Click handler function             |
| `className`   | `string`                           | -           | Additional CSS classes             |

### MenuShortcut Props

| Prop       | Type     | Default | Description                |
| ---------- | -------- | ------- | -------------------------- |
| `shortcut` | `string` | -       | Shortcut text to display   |
| `class`    | `string` | -       | Additional CSS classes     |
