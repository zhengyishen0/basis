## API Reference

### Dialog

Main container with state management and keyboard handling.

| Prop               | Type      | Default                                            | Description                                      |
| ------------------ | --------- | -------------------------------------------------- | ------------------------------------------------ |
| `closeable`        | `boolean` | `true`                                             | Whether dialog can be closed with Escape key     |
| `lockScroll`       | `boolean` | `true`                                             | Prevent body scrolling when dialog is open       |
| `keyboardShortcut` | `string`  | -                                                  | Keyboard shortcut to open dialog (e.g., "cmd+k") |
| `autoFocus`        | `boolean` | `false`                                            | Auto-focus first input/button when opened        |
| `focusSelector`    | `string`  | `'input, button, [tabindex]:not([tabindex="-1"])'` | CSS selector for auto-focus element              |

### DialogContent

Content wrapper with backdrop, animations, and positioning variants.

| Prop       | Type                                                                 | Default    | Description        |
| ---------- | -------------------------------------------------------------------- | ---------- | ------------------ |
| `variant`  | `'center' \, 'left' \, 'right' \, 'top' \, 'bottom' \, 'fullscreen'` | `'center'` | Dialog positioning |
| `backdrop` | `'dark' \, 'light' \, 'blur' \, 'none'`                              | `'dark'`   | Backdrop style     |

### DialogHeader

Header section with optional close button and title content.

| Prop        | Type      | Default | Description                           |
| ----------- | --------- | ------- | ------------------------------------- |
| `closeable` | `boolean` | `true`  | Show close button in top-right corner |

### DialogFooter

Footer section with flexible button layouts.

| Prop      | Type                                        | Default | Description             |
| --------- | ------------------------------------------- | ------- | ----------------------- |
| `justify` | `'start' \, 'end' \, 'center' \, 'between'` | `'end'` | Button alignment        |
| `gap`     | `'sm' \, 'md' \, 'lg'`                      | `'sm'`  | Spacing between buttons |

### DialogTrigger

Trigger element that opens the dialog when clicked.

| Prop      | Type                                            | Default     | Description  |
| --------- | ----------------------------------------------- | ----------- | ------------ |
| `variant` | `'default' \, 'button' \, 'outline' \, 'ghost'` | `'default'` | Visual style |
