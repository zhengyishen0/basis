## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `images` | `Array<ImageObject>` | **Required** | Array of image objects |
| `autoPlayInterval` | `number` | - | Autoplay interval in milliseconds |
| `aspectRatio` | `'auto' | '1/1' | '16/9' | '4/3' | '3/2'` | `'16/9'` | Image aspect ratio |
| `size` | `'sm' | 'md' | 'lg' | 'xl' | 'full'` | `'md'` | Carousel size |
| `class` | `string` | - | Additional CSS classes |

### ImageObject Type

```typescript
interface ImageObject {
    src: string;        // Image source URL (required)
    alt?: string;       // Alt text for accessibility
    thumbnail?: string; // Thumbnail URL (future use)
    caption?: string;   // Caption text overlay
}
```

### Alpine.js State

The carousel creates the following Alpine.js data structure:

```javascript
{
    currentSlide: 0,            // Current active slide index
    images: [...],              // Array of image objects
    autoPlayInterval: 3000,     // Autoplay interval in ms
    autoPlayEnabled: true,      // Whether autoplay is enabled
    autoPlayTimer: null,        // Timer reference
    loop: true,                 // Whether to loop slides
    
    // Methods
    init() { ... },            // Initialize autoplay
    startAutoPlay() { ... },   // Start autoplay timer
    stopAutoPlay() { ... },    // Stop autoplay timer
    next() { ... },            // Go to next slide
    prev() { ... },            // Go to previous slide
    goToSlide(index) { ... }   // Go to specific slide
}
```

### Navigation Features

#### Arrow Navigation
- Left/right arrow buttons appear on hover
- Automatically hidden on mobile-first approach
- Smooth hover transitions with opacity changes

#### Dot Navigation  
- Dot indicators at bottom center
- Click any dot to jump to that slide
- Active dot highlighted with different opacity

#### Slide Counter
- Optional slide counter in top-right corner
- Shows current slide / total slides format
- Hidden by default, visible on hover

### Autoplay Behavior

#### Automatic Playback
- Starts automatically when `autoPlayInterval` is provided
- Pauses on mouse enter, resumes on mouse leave
- Loops infinitely through all slides

#### Manual Control
- Manual navigation stops autoplay temporarily
- Autoplay resumes after mouse leave
- Can be permanently disabled by clearing timer

### Transitions & Effects

#### Fade Transition
- Smooth opacity-based transitions
- 1-second duration with ease-in-out timing
- Overlapping slides with z-index management

#### Caption Overlay
- Gradient background for text readability
- Positioned at bottom of image
- Only shown when caption text is provided

### Accessibility

#### Keyboard Support
- Arrow keys for navigation (when focused)
- Tab navigation to controls
- Enter/Space to activate buttons

#### Screen Reader Support
- Proper alt text for all images
- ARIA labels for navigation buttons
- Semantic button elements

### Advanced Usage

#### Dynamic Image Updates

```astro
<div x-data="{ 
    galleryImages: initialImages,
    updateGallery(newImages) {
        this.galleryImages = newImages;
    }
}">
    <Carousel :images="galleryImages" />
    <button x-on:click="updateGallery(newImageArray)">
        Change Gallery
    </button>
</div>
```

#### Custom Event Handling

```astro
<div x-data="{ 
    currentIndex: 0,
    handleSlideChange(index) {
        this.currentIndex = index;
        // Custom logic here
    }
}">
    <Carousel 
        images={images}
        x-on:slide-changed="handleSlideChange($event.detail.index)"
    />
</div>
```

#### Responsive Breakpoints

The carousel automatically adjusts for different screen sizes:

- **Mobile**: Simplified controls, larger touch targets
- **Tablet**: Standard navigation with hover states  
- **Desktop**: Full feature set with hover effects
