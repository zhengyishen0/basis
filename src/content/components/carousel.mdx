## API Reference

| Prop               | Type                                  | Default      | Description                       |
| ------------------ | ------------------------------------- | ------------ | --------------------------------- |
| `images`           | `Array<ImageObject>`                  | **Required** | Array of image objects            |
| `autoPlayInterval` | `number`                              | -            | Autoplay interval in milliseconds |
| `aspectRatio`      | `'auto', '1/1', '16/9', '4/3', '3/2'` | `'16/9'`     | Image aspect ratio                |
| `size`             | `'sm', 'md', 'lg'`                    | `'md'`       | Carousel size                     |
| `class`            | `string`                              | -            | Additional CSS classes            |

### ImageObject

```typescript
interface ImageObject {
    src: string; // Image source URL (required)
    alt?: string; // Alt text for accessibility
    caption?: string; // Caption text overlay
}
```
