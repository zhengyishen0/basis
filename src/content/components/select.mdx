## Overview

The Select component provides a dropdown selection interface built on top of the Popup component. Features keyboard navigation, smart positioning, grouping support, and seamless Alpine.js integration for reactive behavior.

## Components

### Select

The root component that manages select state and provides context for child components.

| Prop        | Type     | Default | Description            |
| ----------- | -------- | ------- | ---------------------- |
| `className` | `string` | -       | Additional CSS classes |

#### Alpine.js State

Automatically provides the following reactive state:

- `selectedValue`: Currently selected option value
- `selectedLabel`: Currently selected option label

### SelectTrigger

The clickable element that triggers the dropdown. Built on PopupTrigger.

| Prop        | Type                                   | Default     | Description            |
| ----------- | -------------------------------------- | ----------- | ---------------------- |
| `variant`   | `'default', 'button', 'ghost', 'link'` | `'default'` | Visual style variant   |
| `className` | `string`                               | -           | Additional CSS classes |

| Slot    | Description                                         |
| ------- | --------------------------------------------------- |
| Default | Trigger content (typically selected value and icon) |

### SelectContent

The dropdown content container with smooth animations. Built on PopupContent.

| Prop        | Type                               | Default     | Description            |
| ----------- | ---------------------------------- | ----------- | ---------------------- |
| `theme`     | `'default', 'dark', 'minimal'`     | `'default'` | Visual theme           |
| `placement` | `'top', 'bottom', 'left', 'right'` | `'bottom'`  | Dropdown position      |
| `role`      | `string`                           | `'dialog'`  | ARIA role attribute    |
| `className` | `string`                           | -           | Additional CSS classes |

| Slot    | Description                                        |
| ------- | -------------------------------------------------- |
| Default | Dropdown content (SelectItems, SelectGroups, etc.) |

### SelectItem

Individual selectable option with hover and selected states.

| Prop       | Type      | Default | Description                    |
| ---------- | --------- | ------- | ------------------------------ |
| `value`    | `string`  | -       | **Required.** Option value     |
| `label`    | `string`  | `value` | Display label for the option   |
| `disabled` | `boolean` | `false` | Whether the option is disabled |
| `class`    | `string`  | -       | Additional CSS classes         |

| Slot    | Description                                    |
| ------- | ---------------------------------------------- |
| Default | Custom option content (overrides `label` prop) |

### SelectItems

Template component for rendering dynamic options using Alpine.js `x-for` directive.

| Prop    | Type     | Default        | Description                              |
| ------- | -------- | -------------- | ---------------------------------------- |
| `x-for` | `string` | -              | **Required.** Alpine.js x-for expression |
| `x-key` | `string` | `'item.value'` | Key for x-for iteration                  |
| `class` | `string` | -              | Additional CSS classes                   |

### SelectGroup

Container for grouping related options with an optional heading.

| Prop        | Type     | Default | Description                            |
| ----------- | -------- | ------- | -------------------------------------- |
| `x-text`    | `string` | -       | Dynamic text binding for group heading |
| `className` | `string` | -       | Additional CSS classes                 |

| Slot    | Description                 |
| ------- | --------------------------- |
| Default | Group content (SelectItems) |

### SelectGroups

Template component for rendering dynamic groups using Alpine.js `x-for` directive.

| Prop        | Type     | Default      | Description                              |
| ----------- | -------- | ------------ | ---------------------------------------- |
| `x-for`     | `string` | -            | **Required.** Alpine.js x-for expression |
| `x-key`     | `string` | `'group.id'` | Key for x-for iteration                  |
| `x-text`    | `string` | -            | Dynamic text binding for group headings  |
| `className` | `string` | -            | Additional CSS classes                   |

## Alpine.js Integration

The Select component automatically manages state and provides the following reactive properties:

- `selectedValue`: Current selected option value
- `selectedLabel`: Current selected option label
- `close()`: Method to close the dropdown

### Multi-Select Simulation

```astro
<div x-data="{
    selectedItems: [],
    availableItems: [
        { value: 'js', label: 'JavaScript' },
        { value: 'py', label: 'Python' },
        { value: 'rust', label: 'Rust' }
    ],
    toggleItem(value, label) {
        const index = this.selectedItems.findIndex(item => item.value === value);
        if (index > -1) {
            this.selectedItems.splice(index, 1);
        } else {
            this.selectedItems.push({ value, label });
        }
    },
    isSelected(value) {
        return this.selectedItems.some(item => item.value === value);
    }
}">
    <div class="space-y-2">
        <div x-show="selectedItems.length > 0" class="flex flex-wrap gap-1">
            <template x-for="item in selectedItems" x-key="item.value">
                <span class="inline-flex items-center gap-1 px-2 py-1 bg-primary text-primary-foreground text-xs rounded">
                    <span x-text="item.label"></span>
                    <button x-on:click="toggleItem(item.value, item.label)" class="hover:bg-primary-foreground/20 rounded">
                        <Icon name="lucide:x" class="w-3 h-3" />
                    </button>
                </span>
            </template>
        </div>

        <Select>
            <SelectTrigger variant="button" class="w-full justify-between">
                <span x-text="selectedItems.length > 0 ? selectedItems.length + ' selected' : 'Select languages'"></span>
                <Icon name="lucide:chevron-down" class="w-4 h-4" />
            </SelectTrigger>
            <SelectContent role="listbox">
                <template x-for="item in availableItems" x-key="item.value">
                    <div
                        x-on:click="toggleItem(item.value, item.label)"
                        class="flex items-center gap-2 px-3 py-2 hover:bg-accent cursor-pointer"
                        x-bind:class="isSelected(item.value) ? 'bg-accent text-accent-foreground' : ''"
                    >
                        <div class="w-4 h-4 flex items-center justify-center">
                            <Icon x-show="isSelected(item.value)" name="lucide:check" class="w-3 h-3" />
                        </div>
                        <span x-text="item.label"></span>
                    </div>
                </template>
            </SelectContent>
        </Select>
    </div>
</div>
```

### Searchable Select

```astro
<div x-data="{
    selectedValue: '',
    selectedLabel: 'Select option',
    searchQuery: '',
    allOptions: [
        { value: 'apple', label: 'Apple' },
        { value: 'banana', label: 'Banana' },
        { value: 'cherry', label: 'Cherry' },
        { value: 'date', label: 'Date' },
        { value: 'elderberry', label: 'Elderberry' }
    ],
    get filteredOptions() {
        return this.allOptions.filter(option =>
            option.label.toLowerCase().includes(this.searchQuery.toLowerCase())
        );
    }
}">
    <Select>
        <SelectTrigger variant="button" class="w-full justify-between">
            <span x-text="selectedLabel" class="flex-1 text-left"></span>
            <Icon name="lucide:chevron-down" class="w-4 h-4" />
        </SelectTrigger>
        <SelectContent role="listbox">
            <div class="p-2">
                <input
                    type="text"
                    x-model="searchQuery"
                    placeholder="Search options..."
                    class="w-full px-2 py-1 text-sm border rounded focus:outline-none focus:ring-1 focus:ring-primary"
                    x-on:click="$event.stopPropagation()"
                />
            </div>
            <div x-show="filteredOptions.length === 0" class="px-3 py-2 text-sm text-muted-foreground">
                No options found
            </div>
            <template x-for="option in filteredOptions" x-key="option.value">
                <div
                    x-on:click="selectedValue = option.value; selectedLabel = option.label; close()"
                    class="px-3 py-2 hover:bg-accent cursor-pointer text-sm"
                    x-bind:class="selectedValue === option.value ? 'bg-accent text-accent-foreground' : ''"
                >
                    <span x-text="option.label"></span>
                </div>
            </template>
        </SelectContent>
    </Select>
</div>
```
