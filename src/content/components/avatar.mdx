## Overview

A flexible avatar component that displays user profile images with automatic fallback to initials or custom content. Supports multiple sizes, shapes, variants, and can be rendered as a clickable link.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | - | Image source URL |
| `alt` | `string` | - | Image alt text |
| `fallback` | `string` | - | Fallback text when image fails to load |
| `href` | `string` | - | URL to link to (renders as anchor) |
| `variant` | `'default', 'secondary', 'destructive', 'outline', 'ghost'` | `'ghost'` | Semantic color variant |
| `size` | `'xs', 'sm', 'md', 'lg', 'xl'` | `'md'` | Avatar size |
| `shape` | `'rectangle', 'rounded', 'pill'` | `'pill'` | Border radius shape |
| `elevation` | `'none', 'flat', 'elevated'` | `'none'` | Shadow elevation level |
| `imageSize` | `'full', '3/4', '2/3', '1/2'` | `'full'` | Size of image within avatar |
| `className` | `string` | - | Additional CSS classes |

### Slots

| Slot | Description |
|------|-------------|
| `default` | Custom content when no src or fallback provided |

### Behavior

#### Image Loading
- Displays image when `src` is provided and loads successfully
- Falls back to `fallback` text when image fails to load
- Shows slot content when neither `src` nor `fallback` is provided

#### Link Behavior  
- Renders as `<a>` tag when `href` is provided
- Renders as `<div>` tag otherwise
- Maintains all styling and functionality in both cases

#### Fallback Display
- Displays fallback text with muted background and foreground colors
- Centers text both horizontally and vertically
- Uses medium font weight and small text size

### Alpine.js Integration

Avatar supports all Alpine.js directives as pass-through props:

```astro
{/* Click handling */}
<Avatar 
    x-on:click="handleAvatarClick"
    fallback="JS"
/>

{/* Dynamic properties */}
<Avatar 
    x-bind:src="userAvatar"
    x-bind:alt="userName"
    x-bind:fallback="userInitials"
/>

{/* Conditional styling */}
<Avatar 
    x-bind:class="{ 'ring-2 ring-primary': isSelected }"
    src="/avatar.jpg"
/>
```

### Accessibility

- Proper `alt` attributes for screen readers
- Semantic HTML structure (`<a>` vs `<div>`)
- Focus states for clickable avatars
- Color contrast meets accessibility standards
