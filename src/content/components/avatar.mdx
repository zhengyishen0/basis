## API Reference

| Prop        | Type                                                        | Default   | Description                            |
| ----------- | ----------------------------------------------------------- | --------- | -------------------------------------- |
| `src`       | `string`                                                    | -         | Image source URL                       |
| `alt`       | `string`                                                    | -         | Image alt text                         |
| `fallback`  | `string`                                                    | -         | Fallback text when image fails to load |
| `href`      | `string`                                                    | -         | URL to link to (renders as anchor)     |
| `variant`   | `'default', 'secondary', 'destructive', 'outline', 'ghost'` | `'ghost'` | Semantic color variant                 |
| `size`      | `'xs', 'sm', 'md', 'lg', 'xl'`                              | `'md'`    | Avatar size                            |
| `shape`     | `'rectangle', 'rounded', 'pill'`                            | `'pill'`  | Border radius shape                    |
| `elevation` | `'none', 'flat', 'elevated'`                                | `'none'`  | Shadow elevation level                 |
| `imageSize` | `'full', '3/4', '2/3', '1/2'`                               | `'full'`  | Size of image within avatar            |
| `className` | `string`                                                    | -         | Additional CSS classes                 |

| Slot      | Description                                     |
| --------- | ----------------------------------------------- |
| `default` | Custom content when no src or fallback provided |
