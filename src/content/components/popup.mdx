## Overview

The Popup system provides lightweight components for creating tooltips, popovers, and contextual content. Built with Alpine.js, it supports multiple trigger methods and automatic positioning.

## API Reference

### Popup

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `trigger` | `'click' \| 'hover' \| 'focus' \| 'rightclick' \| 'manual'` | `'click'` | How the popup is triggered |
| `delay` | `number` | `0` | Delay in ms before showing popup |
| `className` | `string` | - | Additional CSS classes |

### PopupContent

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `theme` | `'default' \| 'dark' \| 'minimal'` | `'default'` | Visual theme |
| `placement` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'bottom'` | Popup positioning |
| `className` | `string` | - | Additional CSS classes |

### PopupTrigger

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'button' \| 'ghost' \| 'outline' \| 'link'` | `'default'` | Visual style |
| `className` | `string` | - | Additional CSS classes |
