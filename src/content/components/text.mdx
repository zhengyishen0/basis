## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `TextVariant` | `'body'` | Typography variant (see variants table) |
| `state` | `'active', 'completed', 'pending', 'disabled'` | - | Visual state modifier |
| `class` | `string` | - | Additional CSS classes |

### Text Variants

#### Hero Variants

| Variant | Element | Description | Font Size | Font Weight |
|---------|---------|-------------|-----------|-------------|
| `hero-title` | `h1` | Main hero headline | `4xl-6xl` responsive | `bold` |
| `hero-subtitle` | `h3` | Hero subheading | `xl-2xl` responsive | `semibold` |
| `hero-description` | `p` | Hero description | `lg-xl` responsive | `normal` |

#### Card Variants

| Variant | Element | Description | Font Size | Font Weight |
|---------|---------|-------------|-----------|-------------|
| `card-title` | `h2` | Card headline | `xl` | `semibold` |
| `card-subtitle` | `h3` | Card subheading | `lg` | `medium` |
| `card-description` | `p` | Card description | `base` | `normal` |
| `card-content` | `p` | Card body content | `sm` | `normal` |

#### Page Structure

| Variant | Element | Description | Font Size | Font Weight |

### State Variants

| State | Description | Visual Effect |
|-------|-------------|---------------|
| `active` | Currently active/selected | Enhanced contrast, medium weight |
| `completed` | Task completed | Muted color, line-through |
| `pending` | Task pending | Muted color, italic |
| `disabled` | Disabled/unavailable | Muted color, reduced opacity |

### Alpine.js Integration

#### Dynamic States
```astro
<div x-data="{ taskState: 'pending' }">
    <Text variant="body" x-bind:state="taskState">
        Task Status
    </Text>
    <button x-on:click="taskState = 'completed'">
        Complete Task
    </button>
</div>
```

#### Conditional Variants
```astro
<div x-data="{ isImportant: false }">
    <Text 
        :variant="isImportant ? 'hero-title' : 'section-title'"
        x-on:click="isImportant = !isImportant"
    >
        Toggle Importance
    </Text>
</div>
```

#### State-Based Styling
```astro
<Text 
    variant="body"
    x-bind:class="{
        'text-foreground font-medium': $el.dataset.state === 'active',
        'text-muted-foreground line-through': $el.dataset.state === 'completed',
        'text-muted-foreground italic': $el.dataset.state === 'pending',
        'text-muted-foreground opacity-60': $el.dataset.state === 'disabled'
    }"
>
    State-aware text
</Text>
```

#### Semantic Usage
- Use appropriate variants for content hierarchy
- Maintain consistent heading levels
- Choose elements that match content meaning

- Minimal CSS output through CVA
- No JavaScript unless Alpine.js features used
- Optimized for static generation

#### Accessibility
- Always maintain proper heading hierarchy
- Provide sufficient color contrast
- Use semantic elements appropriately
