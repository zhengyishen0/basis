## Overview

A comprehensive text component that automatically renders appropriate HTML elements based on semantic variants. Includes extensive typography scales for hero sections, cards, pages, features, and interactive elements with Alpine.js state support.

## API Reference

| Prop      | Type                                           | Default  | Description                             |
| --------- | ---------------------------------------------- | -------- | --------------------------------------- |
| `variant` | `TextVariant`                                  | `'body'` | Typography variant (see variants table) |
| `state`   | `'active', 'completed', 'pending', 'disabled'` | -        | Visual state modifier                   |
| `class`   | `string`                                       | -        | Additional CSS classes                  |

### State Variants

| State       | Description               | Visual Effect                    |
| ----------- | ------------------------- | -------------------------------- |
| `active`    | Currently active/selected | Enhanced contrast, medium weight |
| `completed` | Task completed            | Muted color, line-through        |
| `pending`   | Task pending              | Muted color, italic              |
| `disabled`  | Disabled/unavailable      | Muted color, reduced opacity     |

### Semantic HTML Elements

The component automatically chooses appropriate HTML elements based on variants:

```javascript
// Element mapping examples
'hero-title' → '<h1>'
'card-title' → '<h2>'
'hero-description' → '<p>'
'testimonial-quote' → '<blockquote>'
'stats-number' → '<strong>'
'body-small' → '<small>'
```

### Alpine.js Integration

#### Dynamic States

```astro
<div x-data="{ taskState: 'pending' }">
    <Text variant="body" x-bind:state="taskState">
        Task Status
    </Text>
    <button x-on:click="taskState = 'completed'">
        Complete Task
    </button>
</div>
```

#### Conditional Variants

```astro
<div x-data="{ isImportant: false }">
    <Text
        :variant="isImportant ? 'hero-title' : 'section-title'"
        x-on:click="isImportant = !isImportant"
    >
        Toggle Importance
    </Text>
</div>
```

#### State-Based Styling

```astro
<Text
    variant="body"
    x-bind:class="{
        'text-foreground font-medium': $el.dataset.state === 'active',
        'text-muted-foreground line-through': $el.dataset.state === 'completed',
        'text-muted-foreground italic': $el.dataset.state === 'pending',
        'text-muted-foreground opacity-60': $el.dataset.state === 'disabled'
    }"
>
    State-aware text
</Text>
```
