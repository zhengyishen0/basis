## API Reference

| Prop          | Type                                              | Default       | Description                 |
| ------------- | ------------------------------------------------- | ------------- | --------------------------- |
| `src`         | `string`                                          | **Required**  | Image source URL            |
| `alt`         | `string`                                          | `''`          | Alt text for accessibility  |
| `aspectRatio` | `'auto', '1/1', '16/9', '4/3', '3/2'`             | `'auto'`      | Aspect ratio constraint     |
| `objectFit`   | `'cover', 'contain', 'fill', 'none', 'scaleDown'` | `'cover'`     | How image fits container    |
| `shape`       | `'rectangle', 'rounded', 'pill'`                  | `'rectangle'` | Border radius style         |
| `overlay`     | `string`                                          | -             | CSS classes for overlay div |
| `loading`     | `'lazy', 'eager'`                                 | `'lazy'`      | Image loading behavior      |
| `class`       | `string`                                          | -             | Additional CSS classes      |

| Slot      | Description                          |
| --------- | ------------------------------------ |
| `default` | Content overlaid on top of the image |
