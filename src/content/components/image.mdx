## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | **Required** | Image source URL |
| `alt` | `string` | `''` | Alt text for accessibility |
| `aspectRatio` | `'auto' | '1/1' | '16/9' | '4/3' | '3/2'` | `'auto'` | Aspect ratio constraint |
| `objectFit` | `'cover' | 'contain' | 'fill' | 'none' | 'scaleDown'` | `'cover'` | How image fits container |
| `shape` | `'rectangle' | 'rounded' | 'pill'` | `'rectangle'` | Border radius style |
| `overlay` | `string` | - | CSS classes for overlay div |
| `loading` | `'lazy' | 'eager'` | `'lazy'` | Image loading behavior |
| `class` | `string` | - | Additional CSS classes |

### Slots

| Slot | Description |
|------|-------------|
| `default` | Content overlaid on top of the image |

### Aspect Ratio Behavior

#### Available Ratios
- `auto`: Natural image dimensions
- `1/1`: Perfect square (1:1)
- `16/9`: Widescreen video format
- `4/3`: Traditional photo format  
- `3/2`: Wide photo format

#### Implementation
Uses Tailwind's aspect ratio utilities:
```css
aspect-square    /* 1:1 */
aspect-video     /* 16:9 */  
aspect-[4/3]     /* 4:3 */
aspect-[3/2]     /* 3:2 */
```

### Object Fit Options

#### Cover (Default)
- Image covers entire container
- May crop parts of image
- Maintains aspect ratio

#### Contain
- Entire image visible
- May leave empty space
- Maintains aspect ratio

#### Fill  
- Fills container exactly
- May distort image
- Ignores aspect ratio

#### None
- Image keeps natural size
- May overflow or leave space
- No scaling applied

#### Scale Down
- Like `none` or `contain`
- Only scales down, never up
- Prevents pixelation

### Shape Variants

#### Rectangle
- No border radius
- Sharp corners
- Default geometric shape

#### Rounded  
- Medium border radius
- Softened corners
- Balanced appearance

#### Pill
- Maximum border radius
- Fully rounded edges
- Circular when square

### Alpine.js Integration

#### Dynamic Sources
```astro
<div x-data="{ imageSrc: '/default.jpg' }">
    <Image x-bind:src="imageSrc" alt="Dynamic image" />
    <button x-on:click="imageSrc = '/new-image.jpg'">
        Change Image
    </button>
</div>
```

#### Image Galleries
```astro
<div x-data="{ 
    currentIndex: 0,
    images: ['/img1.jpg', '/img2.jpg', '/img3.jpg']
}">
    <Image 
        x-bind:src="images[currentIndex]"
        alt="Gallery image"
    />
    <button x-on:click="currentIndex = (currentIndex + 1) % images.length">
        Next
    </button>
</div>
```

#### Loading States
```astro
<div x-data="{ loading: true }">
    <Image 
        src="/image.jpg"
        alt="Loading example"
        x-on:load="loading = false"
        x-bind:class="{ 'opacity-50': loading }"
    />
    <div x-show="loading">Loading...</div>
</div>
```

### Performance Optimization

#### Lazy Loading
- Default behavior for performance
- Images load when entering viewport
- Reduces initial page load time

#### Eager Loading  
- For above-the-fold images
- Critical hero images
- LCP optimization

#### Responsive Considerations
```astro
{/* Responsive sizing */}
<Image 
    src="/image.jpg"
    alt="Responsive image"
    class="w-full sm:w-1/2 md:w-1/3 lg:w-1/4"
/>

{/* Different images for different sizes */}
<div x-data="{ 
    getImageSrc() {
        return window.innerWidth > 768 ? '/large.jpg' : '/small.jpg';
    }
}">
    <Image x-bind:src="getImageSrc()" alt="Responsive source" />
</div>
```

### Accessibility

#### Alt Text
- Always provide descriptive alt text
- Empty alt for decorative images
- Context-appropriate descriptions

#### Keyboard Navigation
- Focusable when interactive
- Proper tab order
- Clear focus indicators

#### Screen Reader Support
- Semantic image elements
- Proper ARIA attributes
- Meaningful descriptions
