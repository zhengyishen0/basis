## Overview

Cards are surfaces that display content and actions on a single topic. They should be easy to scan for relevant and actionable information.

## API Reference

| Component     | Props                | Description                               |
| ------------- | -------------------- | ----------------------------------------- |
| `Card`        | `class`, `elevation` | Container wrapper with padding and border |
| `CardHeader`  | `class`              | Header section with title and description |
| `CardContent` | `class`              | Main content area                         |
| `CardFooter`  | `class`              | Footer with actions                       |
