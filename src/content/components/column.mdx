## Overview

The Column component is a layout-driven component that arranges elements vertically. It focuses purely on positioning through `justify` and `align` properties, with no gap or padding properties. Content sizing is determined by child elements.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `overflow` | `'auto' | 'fixed' | 'expand' | 'wrap' | 'show'` | `'expand'` | Controls overflow behavior |
| `class` | `string` | - | Additional CSS classes |
| `...alpineProps` | `any` | - | Alpine.js attributes (x-data, x-show, etc.) |

### Overflow Behaviors

- **`expand`** (default): No constraints, container grows with content
- **`auto`**: Vertical scrolling when content overflows
- **`fixed`**: Content is clipped at container boundaries  
- **`show`**: Content can spill outside container bounds

### Slots

| Slot | Description |
|------|-------------|
| default | Content to be arranged vertically |

## Design Principles

Column is a **layout-driven component** that:
- Focuses on positioning with flexbox justify and align properties
- Has no gap or padding properties (use CSS classes if needed)
- Takes available space from parent container
- Handles static layout positioning
- Default overflow is `expand` for natural growth
