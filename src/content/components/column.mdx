## Overview

A simple layout component that arranges elements vertically using flexbox.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `overflow` | `'auto' \, 'fixed' \, 'expand'` | `'expand'` | Controls overflow behavior |
| `class` | `string` | - | Additional CSS classes |
| `...alpineProps` | `any` | - | Alpine.js attributes |
