## Overview

Alerts display important messages in a way that attracts user attention without interrupting the user's task.

## Import

<Code 
    code="import Alert from '@/components/ui/feedback/Alert.astro';" 
    language="typescript" 
/>

## Basic Usage

<div class="space-y-4">
    <Alert variant="default">
        <strong>Information:</strong> This is a default alert message.
    </Alert>
    <Alert variant="success">
        <strong>Success!</strong> Your changes have been saved.
    </Alert>
    <Alert variant="warning">
        <strong>Warning:</strong> Please review your input.
    </Alert>
    <Alert variant="destructive">
        <strong>Error:</strong> Something went wrong.
    </Alert>
</div>

<Code 
    code={'<Alert variant="default">Default alert message</Alert>\n<Alert variant="success">Success message</Alert>\n<Alert variant="warning">Warning message</Alert>\n<Alert variant="destructive">Error message</Alert>'}
    language="astro" 
/>

## With Icons and Actions

<Alert variant="success" class="flex items-center justify-between">

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'success' \| 'warning' \| 'destructive'` | `'default'` | Alert style variant |
| `class` | `string` | - | Additional CSS classes |
