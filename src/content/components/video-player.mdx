## Overview

The VideoPlayer component provides a fully-featured video playback experience with custom controls, progress tracking, volume management, and support for both local video files and YouTube content. Built with Alpine.js for reactive state management.

## API Reference

| Prop          | Type                        | Default     | Description                                |
| ------------- | --------------------------- | ----------- | ------------------------------------------ |
| `src`         | `string`                    | -           | Local video file URL (MP4 recommended)     |
| `poster`      | `string`                    | -           | Poster image URL displayed before playback |
| `youtubeId`   | `string`                    | -           | YouTube video ID (alternative to src)      |
| `autoplay`    | `boolean`                   | `false`     | Auto-start video playback                  |
| `loop`        | `boolean`                   | `false`     | Loop video when it ends                    |
| `muted`       | `boolean`                   | `false`     | Start video muted                          |
| `shape`       | `'rectangle' \, 'rounded'`  | `'rounded'` | Corner style of video player               |
| `aspectRatio` | `'16/9' \, '4/3' \, '21/9'` | `'16/9'`    | Video aspect ratio                         |
| `class`       | `string`                    | -           | Additional CSS classes                     |

### Alpine.js State (Local Video Only)

The local video player exposes these reactive properties:

```javascript
{
  playing: false,        // Current playback state
  muted: false,         // Current mute state
  volume: 1,            // Volume level (0-1)
  currentTime: 0,       // Current playback time in seconds
  duration: 0,          // Total video duration in seconds
  buffered: 0,          // Amount of video buffered in seconds
  loading: true,        // Loading state
  showControls: false   // Controls visibility state
}
```

### Methods (Local Video Only)

Access these methods through Alpine.js component data:

```javascript
// Playback control
this.togglePlay()          // Toggle play/pause
this.seek(timeInSeconds)   // Seek to specific time
this.seekToPercentage(%)   // Seek to percentage of duration

// Volume control
this.toggleMute()          // Toggle mute state
this.setVolume(0.5)        // Set volume (0-1)

// Utility
this.formatTime(seconds)   // Format seconds as MM:SS
this.getProgressPercentage() // Get current progress as percentage
this.getBufferedPercentage() // Get buffered amount as percentage
```
