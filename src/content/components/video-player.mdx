## Overview

The VideoPlayer component provides a fully-featured video playback experience with custom controls, progress tracking, volume management, and support for both local video files and YouTube content. Built with Alpine.js for reactive state management.

## Features

- **Dual Source Support**: Local video files and YouTube videos
- **Custom Controls**: Play/pause, progress bar, volume control, time display
- **Responsive Design**: Automatic scaling with aspect ratio preservation
- **Interactive Elements**: Hover controls, smooth animations, gradient overlays
- **Volume Management**: Visual volume slider with mute toggle
- **Progress Tracking**: Visual progress bar with buffering indicator
- **Keyboard Support**: Standard video player keyboard shortcuts
- **Mobile Friendly**: Touch-optimized controls and responsive layout
- **Alpine.js Integration**: Reactive state management and event handling

## Aspect Ratios

The component supports three common video aspect ratios:

- **16:9** - Standard widescreen (YouTube, most modern content)
- **4:3** - Traditional television format
- **21:9** - Ultra-wide cinematic format

## API Reference

### Props

| Prop          | Type                        | Default     | Description                                |
| ------------- | --------------------------- | ----------- | ------------------------------------------ |
| `src`         | `string`                    | -           | Local video file URL (MP4 recommended)     |
| `poster`      | `string`                    | -           | Poster image URL displayed before playback |
| `youtubeId`   | `string`                    | -           | YouTube video ID (alternative to src)      |
| `autoplay`    | `boolean`                   | `false`     | Auto-start video playback                  |
| `loop`        | `boolean`                   | `false`     | Loop video when it ends                    |
| `muted`       | `boolean`                   | `false`     | Start video muted                          |
| `shape`       | `'rectangle' \, 'rounded'`  | `'rounded'` | Corner style of video player               |
| `aspectRatio` | `'16/9' \, '4/3' \, '21/9'` | `'16/9'`    | Video aspect ratio                         |
| `class`       | `string`                    | -           | Additional CSS classes                     |

### Alpine.js State (Local Video Only)

The local video player exposes these reactive properties:

```javascript
{
  playing: false,        // Current playback state
  muted: false,         // Current mute state
  volume: 1,            // Volume level (0-1)
  currentTime: 0,       // Current playback time in seconds
  duration: 0,          // Total video duration in seconds
  buffered: 0,          // Amount of video buffered in seconds
  loading: true,        // Loading state
  showControls: false   // Controls visibility state
}
```

### Methods (Local Video Only)

Access these methods through Alpine.js component data:

```javascript
// Playback control
this.togglePlay()          // Toggle play/pause
this.seek(timeInSeconds)   // Seek to specific time
this.seekToPercentage(%)   // Seek to percentage of duration

// Volume control
this.toggleMute()          // Toggle mute state
this.setVolume(0.5)        // Set volume (0-1)

// Utility
this.formatTime(seconds)   // Format seconds as MM:SS
this.getProgressPercentage() // Get current progress as percentage
this.getBufferedPercentage() // Get buffered amount as percentage
```

### Interactive Player with Events

```astro
<div x-data="{
    videoPlaying: false,
    watchTime: 0
}">
    <VideoPlayer
        src="/videos/tutorial.mp4"
        @alpine-play="videoPlaying = true"
        @alpine-pause="videoPlaying = false"
        @alpine-timeupdate="watchTime = $event.detail.currentTime"
    />

    <div class="mt-4 text-sm text-gray-600">
        <p>Status: <span x-text="videoPlaying ? 'Playing' : 'Paused'"></span></p>
        <p>Watch time: <span x-text="Math.floor(watchTime)"></span> seconds</p>
    </div>
</div>
```

## YouTube Integration

When using YouTube videos, the component automatically:

- Embeds the YouTube player with optimal settings
- Enables JavaScript API for programmatic control
- Applies modest branding to reduce YouTube UI clutter
- Disables related videos at the end
- Supports autoplay (subject to browser policies)

```astro
<!-- YouTube video with autoplay -->
<VideoPlayer
    youtubeId="your-video-id"
    autoplay={true}
    aspectRatio="16/9"
/>
```

## Accessibility

The VideoPlayer component includes comprehensive accessibility features:

- **Keyboard Controls**: Space (play/pause), Arrow keys (seek), M (mute)
- **Screen Reader Support**: Proper ARIA labels for all controls
- **Focus Management**: Visible focus indicators on all interactive elements
- **High Contrast**: Controls remain visible in high contrast mode
- **Captions Support**: YouTube videos support closed captions
- **Reduced Motion**: Respects user's motion preferences

## Performance Considerations

### Local Videos

- Use modern video formats (MP4 with H.264)
- Optimize file sizes for web delivery
- Consider video compression and multiple quality options
- Use appropriate poster images to reduce initial load

### YouTube Videos

- Lazy loading is built-in
- No additional JavaScript bundle impact
- Leverages YouTube's global CDN
- Automatic quality adaptation

## Browser Support

**Local Video Features**:

- Chrome 60+, Firefox 55+, Safari 12+, Edge 79+
- Requires support for HTML5 video element
- Progressive enhancement for older browsers

**YouTube Integration**:

- Works in all browsers that support iframes
- Enhanced features require JavaScript enabled

## Troubleshooting

### Video Won't Play

- Check MIME types are configured correctly on your server
- Ensure video format is supported (MP4 recommended)
- Verify CORS headers if serving from different domain

### YouTube Videos Not Loading

- Verify the YouTube video ID is correct
- Check that the video is public and embeddable
- Some videos may have regional restrictions

### Controls Not Responsive

- Ensure Alpine.js is loaded and initialized
- Check browser console for JavaScript errors
- Verify video element is properly referenced
