## Overview

Loading is a simple component for displaying loading states with optional spinner animations. Provides consistent styling and spacing for loading messages.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm', 'md', 'lg'` | `'md'` | Size of the loading component |
| `spinner` | `boolean` | `false` | Shows animated spinner icon |
| `class` | `string` | `undefined` | Additional CSS classes |

### Slots

| Slot | Description |
|------|-------------|
| default | Loading message or content |

### Size Specifications

| Size | Padding | Text Size | Use Case |
|------|---------|-----------|----------|
| `sm` | `py-6` | `text-sm` | Compact loading states |
| `md` | `py-12` | `text-lg` | Standard loading states |
| `lg` | `py-16` | `text-xl` | Full page loading states |
