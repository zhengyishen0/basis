## Overview

Progress is a visual indicator component that displays the completion progress of a task. It supports different variants, animation states, striped patterns, auto-increment functionality, and Alpine.js integration for dynamic updates.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | `0` | Current progress value |
| `max` | `number` | `100` | Maximum progress value |
| `variant` | `'default', 'secondary', 'destructive', 'success', 'warning', 'info'` | `'default'` | Progress bar color variant |
| `animated` | `boolean` | `false` | Enables pulse animation |
| `striped` | `boolean` | `false` | Adds diagonal striped pattern |
| `showLabel` | `boolean` | `false` | Shows progress percentage label |
| `autoIncrement` | `boolean` | `false` | Automatically increments progress |
| `incrementInterval` | `number` | `100` | Auto-increment interval in milliseconds |
| `class` | `string` | `undefined` | Additional CSS classes |

### Alpine.js Integration

The Progress component provides several Alpine.js data properties and methods:

| Property | Type | Description |
|----------|------|-------------|
| `progress` | `number` | Current progress value (reactive) |
| `max` | `number` | Maximum progress value |
| `percentage` | `number` | Computed percentage (0-100) |
| `resetProgress()` | `function` | Resets progress to 0 and restarts auto-increment |
| `startAutoIncrement()` | `function` | Starts the auto-increment timer |

### Usage Examples

```astro
{/* Basic usage */}
<Progress value={50} showLabel={true} />

{/* Different variants */}
<Progress value={75} variant="success" showLabel={true} />
<Progress value={25} variant="warning" showLabel={true} />

{/* With visual effects */}
<Progress 
  value={80} 
  striped={true} 
  animated={true} 
  showLabel={true} 
/>

{/* Auto-incrementing */}
<Progress 
  autoIncrement={true}
  incrementInterval={200}
  variant="info"
  showLabel={true}
/>

{/* Dynamic with Alpine.js */}
<div x-data="{ progress: 30 }">
  <Progress x-bind:value="progress" showLabel={true} />
  <button x-on:click="progress += 10">Increase</button>
</div>
```

### Features

- **CVA Variants**: Full shadcn-style variant system with theme colors
- **Visual Effects**: Striped patterns and pulse animations
- **Auto-increment**: Built-in timer functionality with reset controls
- **Alpine.js Ready**: Reactive data binding and method exposure
- **Responsive**: Adapts to container width
- **Accessible**: Semantic progress element with proper ARIA attributes
- **Customizable**: Full theming support through CSS custom properties

### Styling

The Progress component uses CSS custom properties and Tailwind classes:

```css
/* Custom striped pattern */
.bg-stripes {
  background-image: repeating-linear-gradient(
    45deg,
    rgba(255, 255, 255, 0.1) 0,
    rgba(255, 255, 255, 0.1) 0.5rem,
    transparent 0.5rem,
    transparent 1rem
  );
}
```

The component supports all theme colors defined in your `tailwind.config.js` file.
