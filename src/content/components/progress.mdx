## Overview

Progress is a visual indicator component that displays the completion progress of a task. It supports different variants, animation states, striped patterns, auto-increment functionality, and Alpine.js integration for dynamic updates.

## API Reference

| Prop                | Type                                                                  | Default     | Description                             |
| ------------------- | --------------------------------------------------------------------- | ----------- | --------------------------------------- |
| `value`             | `number`                                                              | `0`         | Current progress value                  |
| `max`               | `number`                                                              | `100`       | Maximum progress value                  |
| `variant`           | `'default', 'secondary', 'destructive', 'success', 'warning', 'info'` | `'default'` | Progress bar color variant              |
| `animated`          | `boolean`                                                             | `false`     | Enables pulse animation                 |
| `striped`           | `boolean`                                                             | `false`     | Adds diagonal striped pattern           |
| `showLabel`         | `boolean`                                                             | `false`     | Shows progress percentage label         |
| `autoIncrement`     | `boolean`                                                             | `false`     | Automatically increments progress       |
| `incrementInterval` | `number`                                                              | `100`       | Auto-increment interval in milliseconds |
| `class`             | `string`                                                              | `undefined` | Additional CSS classes                  |

### Alpine.js Integration

The Progress component provides several Alpine.js data properties and methods:

| Property               | Type       | Description                                      |
| ---------------------- | ---------- | ------------------------------------------------ |
| `progress`             | `number`   | Current progress value (reactive)                |
| `max`                  | `number`   | Maximum progress value                           |
| `percentage`           | `number`   | Computed percentage (0-100)                      |
| `resetProgress()`      | `function` | Resets progress to 0 and restarts auto-increment |
| `startAutoIncrement()` | `function` | Starts the auto-increment timer                  |
