## Overview

The CopyToClipboard component provides a simple way to copy text to the user's clipboard with visual feedback. It supports both button and icon variants with customizable success/error messages and timeout durations.

<Tabs defaultValue="preview" class="h-auto">
    <TabsList variant="pills" slot="list">

## API Reference

| Prop             | Type               | Default            | Description                                  |
| ---------------- | ------------------ | ------------------ | -------------------------------------------- |
| `text`           | `string`           | -                  | Text to copy to clipboard (required)         |
| `variant`        | `'button', 'icon'` | `'button'`         | Display variant                              |
| `successMessage` | `string`           | `'Copied!'`        | Message shown on successful copy             |
| `errorMessage`   | `string`           | `'Failed to copy'` | Message shown on copy failure                |
| `timeout`        | `number`           | `3000`             | Duration to show notification (milliseconds) |
| `class`          | `string`           | -                  | Additional CSS classes                       |
