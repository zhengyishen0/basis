## Overview

The CopyToClipboard component provides a simple way to copy text to the user's clipboard with visual feedback. It supports both button and icon variants with customizable success/error messages and timeout durations.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | - | Text to copy to clipboard (required) |
| `variant` | `'button', 'icon'` | `'button'` | Display variant |
| `successMessage` | `string` | `'Copied!'` | Message shown on successful copy |
| `errorMessage` | `string` | `'Failed to copy'` | Message shown on copy failure |
| `timeout` | `number` | `3000` | Duration to show notification (milliseconds) |
| `class` | `string` | - | Additional CSS classes |

### Features

- **Clipboard Integration**: Uses the modern Navigator Clipboard API for reliable copying
- **Visual Feedback**: Shows success/error states with animated transitions
- **Button Variant**: Full button with text and icons that change based on state
- **Icon Variant**: Compact icon-only version perfect for code blocks
- **Customizable Messages**: Configure success and error messages
- **Auto-hide Notification**: Configurable timeout for notification visibility
- **Tooltip Support**: Icon variant shows helpful tooltips
- **Accessibility**: Proper focus states and keyboard interaction
- **Alpine.js Integration**: Fully reactive with Alpine.js state management

### Usage Examples

#### In Code Blocks
Perfect for adding copy functionality to code snippets:

```astro
<div class="relative">
    <pre><code>npm install my-package</code></pre>
    <CopyToClipboard 
        text="npm install my-package"
        variant="icon"
        class="absolute top-2 right-2"
    />
</div>
```

#### With Dynamic Content
Use with Alpine.js for dynamic content:

```astro
<div x-data="{ code: 'const x = 42;' }">
    <textarea x-model="code"></textarea>
    <CopyToClipboard 
        x-bind:text="code"
        variant="button"
        successMessage="Code copied!"
    />
</div>
```

#### API Keys or Tokens
For copying sensitive information:

```astro
<CopyToClipboard 
    text="sk-1234567890abcdef"
    variant="button"
    successMessage="API key copied!"
    errorMessage="Failed to copy API key"
    class="text-sm"
/>
```
