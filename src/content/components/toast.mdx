## Overview

Toast is a global notification system that provides temporary, non-intrusive messages to users. It's built with Alpine.js and offers a powerful global API for showing notifications from anywhere in your application. The component handles automatic positioning, stacking, animations, and dismissal.

## API Reference

### Global Functions

The Toast component provides a global `toast()` function that can be called from anywhere:

| Function | Description |
|----------|-------------|
| `toast(message, options?)` | Shows a toast notification with the given message and options |

### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `type` | `'default', 'success', 'info', 'warning', 'danger'` | `'default'` | Toast variant/type |
| `description` | `string` | `undefined` | Additional description text |
| `position` | `'top-left', 'top-center', 'top-right', 'bottom-left', 'bottom-center', 'bottom-right'` | `'top-center'` | Toast position |
| `duration` | `number` | `4000` | Auto-dismiss duration in milliseconds (0 = no auto-dismiss) |
| `html` | `string` | `undefined` | Custom HTML content (overrides message and description) |

### Alpine.js Integration

The Toast component integrates with Alpine.js through:

- **Global Events**: Uses `@toast-show.window` to trigger toasts
- **State Management**: Manages toast queue and positioning
- **Animations**: Smooth enter/exit transitions
- **Interaction**: Hover to expand stacked toasts, click to dismiss

### Usage Examples

```javascript
// Basic usage
toast('Hello world!');

// With type and description
toast('Task completed', {
  type: 'success',
  description: 'Your file has been uploaded successfully.'
});

// Custom position and duration
toast('Connection lost', {
  type: 'danger',
  position: 'bottom-right',
  duration: 0 // Persistent
});

// Custom HTML content
toast('', {
  html: '<strong>Rich content</strong> with <em>HTML</em>',
  type: 'info'
});
```

### Features

- **Global System**: Single component instance handles all toasts
- **Smart Stacking**: Automatic positioning and stacking of multiple toasts
- **Hover Expansion**: Stack expands on hover for better visibility
- **Smooth Animations**: Enter/exit transitions with position-aware directions
- **Auto-dismiss**: Configurable duration with pause on hover
- **Type-based Icons**: Automatic icons for success, error, warning, and info types
- **Responsive**: Adapts to mobile screens with proper spacing
- **Accessible**: Includes close buttons and proper ARIA attributes

### Installation

1. Add the Toast component to your layout (once per page):

```astro
---
import Toast from '@/components/ui/feedback/Toast.astro';
---

<Toast />
```

2. Use the global `toast()` function anywhere in your Alpine.js code:

```html
<button x-on:click="toast('Hello!')">Show Toast</button>
```
