## Overview

The SmartBreadcrumb component automatically generates breadcrumb navigation with intelligent overflow handling. When there are too many items, it shows ellipsis with a dropdown containing the hidden items.

## API Reference

| Prop         | Type                                           | Default      | Description                               |
| ------------ | ---------------------------------------------- | ------------ | ----------------------------------------- |
| `variant`    | `'default' | 'minimal' | 'compact'`          | `'default'`  | Visual style variant                      |
| `size`       | `'sm' | 'md' | 'lg'`                        | `'md'`       | Size of the breadcrumb                    |
| `maxItems`   | `number`                                       | `5`          | Maximum items before showing ellipsis    |
| `separator`  | `'chevron' | 'slash' | 'dot' | 'arrow'`      | `'chevron'`  | Separator icon between items             |
| `overflow`   | `'fixed' | 'wrap' | 'scroll'`               | `'fixed'`    | Overflow handling strategy                |
| `aria-label` | `string`                                       | `'breadcrumb'` | ARIA label for accessibility           |
| `class`      | `string`                                       | -            | Additional CSS classes                    |

### Alpine.js Data Structure

```typescript
{
  items: Array<{
    label: string;    // Display text for the breadcrumb
    href: string;     // URL for the breadcrumb link
  }>
}
```
