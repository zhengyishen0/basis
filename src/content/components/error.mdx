## Overview

Error is a versatile component for displaying error messages, warnings, informational alerts, and success messages with color-coded variants and flexible sizing.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'error', 'warning', 'info', 'success'` | `'error'` | Visual variant/theme |
| `size` | `'sm', 'md', 'lg'` | `'md'` | Size of the component |
| `class` | `string` | `undefined` | Additional CSS classes |

### Slots

| Slot | Description |
|------|-------------|
| default | Error message content |

### Variant Colors

| Variant | Background | Border | Text |
|---------|------------|--------|------|
| `error` | `bg-red-50` | `border-red-200` | `text-red-800` |
| `warning` | `bg-yellow-50` | `border-yellow-200` | `text-yellow-800` |
| `info` | `bg-blue-50` | `border-blue-200` | `text-blue-800` |
| `success` | `bg-green-50` | `border-green-200` | `text-green-800` |

### Size Specifications

| Size | Padding | Text Size |
|------|---------|-----------|
| `sm` | `p-3` | `text-sm` |
| `md` | `p-4` | `text-base` |
| `lg` | `p-6` | `text-lg` |
