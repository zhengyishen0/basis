## Overview

ContextMenu provides a right-click menu that appears when users right-click on an element. It's perfect for providing contextual actions and shortcuts without cluttering the interface.

## Usage Patterns

### File Operations
Perfect for file browsers, project explorers, and any interface where users work with files and folders.

### Text Editing
Essential for rich text editors, code editors, and any text manipulation interface.

### Canvas/Drawing Apps
Ideal for drawing applications, design tools, and any spatial interface where right-click actions make sense.

### Data Tables
Great for providing row-specific actions in tables without cluttering the interface.

## API Reference

### ContextMenu Props

| Prop        | Type                                        | Default       | Description                    |
| ----------- | ------------------------------------------- | ------------- | ------------------------------ |
| `trigger`   | `'click' \| 'rightclick' \| 'hover' \| 'manual'` | `'rightclick'` | How the menu is triggered      |
| `className` | `string`                                    | -             | Additional CSS classes         |

### MenuSubMenu Props

| Prop        | Type     | Default | Description                |
| ----------- | -------- | ------- | -------------------------- |
| `className` | `string` | -       | Additional CSS classes     |

### MenuSubTrigger Props

| Prop        | Type     | Default | Description                |
| ----------- | -------- | ------- | -------------------------- |
| `className` | `string` | -       | Additional CSS classes     |

## Examples

### Simple Context Menu

```astro
<ContextMenu>
    <div slot="trigger">Right-click me</div>
    <div slot="content">
        <MenuItem>Action 1</MenuItem>
        <MenuItem>Action 2</MenuItem>
        <MenuItem disabled>Disabled Action</MenuItem>
    </div>
</ContextMenu>
```

### With Groups and Separators

```astro
<ContextMenu>
    <div slot="trigger">Right-click for grouped menu</div>
    <div slot="content">
        <MenuGroup>
            <MenuItem>Cut</MenuItem>
            <MenuItem>Copy</MenuItem>
            <MenuItem>Paste</MenuItem>
        </MenuGroup>
        <div class="border-t my-1"></div>
        <MenuGroup>
            <MenuItem>Select All</MenuItem>
            <MenuItem>Find</MenuItem>
        </MenuGroup>
    </div>
</ContextMenu>
```

### Multi-level Submenus

```astro
<ContextMenu>
    <div slot="trigger">Complex menu</div>
    <div slot="content">
        <MenuItem>Regular Item</MenuItem>
        <MenuSubMenu>
            <MenuSubTrigger x-ref="submenuTrigger">
                Level 1
                <Icon name="lucide:chevron-right" class="ml-auto" />
            </MenuSubTrigger>
            <div slot="submenu">
                <MenuItem>Level 1 Item</MenuItem>
                <MenuSubMenu>
                    <MenuSubTrigger x-ref="submenuTrigger">
                        Level 2
                        <Icon name="lucide:chevron-right" class="ml-auto" />
                    </MenuSubTrigger>
                    <div slot="submenu">
                        <MenuItem>Level 2 Item</MenuItem>
                    </div>
                </MenuSubMenu>
            </div>
        </MenuSubMenu>
    </div>
</ContextMenu>
```

## Accessibility

- Context menus support keyboard navigation
- Use semantic HTML roles for screen readers
- Include keyboard shortcuts where appropriate
- Ensure sufficient color contrast for menu items
- Support focus management when opening/closing
