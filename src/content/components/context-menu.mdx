## Overview

ContextMenu provides a right-click menu that appears when users right-click on an element. Perfect for providing contextual actions and shortcuts without cluttering the interface.

## API Reference

| Prop        | Type                                        | Default       | Description                    |
| ----------- | ------------------------------------------- | ------------- | ------------------------------ |
| `trigger`   | `'click' \, 'rightclick' \, 'hover' \, 'manual'` | `'rightclick'` | How the menu is triggered      |
| `className` | `string`                                    | -             | Additional CSS classes         |
