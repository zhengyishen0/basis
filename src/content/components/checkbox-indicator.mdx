## States

The CheckboxIndicator supports three primary visual states:

### Unchecked

- Default state with empty checkbox
- Border visible, no fill or icon

### Checked

- Filled state with checkmark icon
- Smooth scale animation on state change

### Indeterminate

- Partial selection state with horizontal line
- Useful for parent checkboxes in hierarchical lists

### Disabled

- Reduced opacity and no interaction
- Maintains visual state but prevents user input

## API Reference

### Props

| Prop            | Type                                                      | Default     | Description                             |
| --------------- | --------------------------------------------------------- | ----------- | --------------------------------------- |
| `checked`       | `boolean`                                                 | `false`     | Checked state of the checkbox           |
| `indeterminate` | `boolean`                                                 | `false`     | Indeterminate state (overrides checked) |
| `disabled`      | `boolean`                                                 | `false`     | Disabled state                          |
| `state`         | `'default' \| 'checked' \| 'indeterminate' \| 'disabled'` | `'default'` | Visual state variant                    |
| `name`          | `string`                                                  | -           | Form input name attribute               |
| `value`         | `string`                                                  | -           | Form input value                        |
| `class`         | `string`                                                  | -           | Additional CSS classes                  |

### Alpine.js Props

When using with Alpine.js, you can pass reactive expressions:

| Prop             | Type     | Description                                  |
| ---------------- | -------- | -------------------------------------------- |
| `:checked`       | `string` | Alpine.js expression for checked state       |
| `:indeterminate` | `string` | Alpine.js expression for indeterminate state |
| `:disabled`      | `string` | Alpine.js expression for disabled state      |

#### Hierarchical Selection

```javascript
// Alpine.js data for parent-child checkboxes
{
    categories: [
        {
            name: 'Frontend',
            items: ['React', 'Vue', 'Angular'],
            selected: []
        }
    ],
    get categoryState(category) {
        if (category.selected.length === 0) return 'unchecked';
        if (category.selected.length === category.items.length) return 'checked';
        return 'indeterminate';
    }
}
```
