## Overview

Empty is a minimalist component for displaying empty states with consistent styling and spacing. Perfect for showing messages when there's no content to display.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm', 'md', 'lg'` | `'md'` | Size of the empty state |
| `class` | `string` | `undefined` | Additional CSS classes |

### Slots

| Slot | Description |
|------|-------------|
| default | Empty state message or content |

### Size Specifications

| Size | Padding | Text Size | Use Case |
|------|---------|-----------|----------|
| `sm` | `py-6` | `text-sm` | Compact spaces, table rows |
| `md` | `py-12` | `text-lg` | Standard containers, cards |
| `lg` | `py-16` | `text-xl` | Full page states, main areas |
