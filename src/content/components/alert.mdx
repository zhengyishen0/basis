## Overview

Alerts display important messages in a way that attracts user attention without interrupting the user's task.

## API Reference

| Prop          | Type                                  | Default     | Description            |
| ------------- | ------------------------------------- | ----------- | ---------------------- |
| `variant`     | `'default', 'outline', 'destructive'` | `'default'` | Alert style variant    |
| `title`       | `string`                              | -           | Alert title text       |
| `description` | `string`                              | -           | Alert description text |
| `dismissible` | `boolean`                             | `false`     | Shows dismiss button   |
| `padding`     | `string`                              | -           | Padding size           |
| `elevation`   | `string`                              | -           | Shadow elevation       |
| `class`       | `string`                              | -           | Additional CSS classes |

| Slot      | Description                                          |
| --------- | ---------------------------------------------------- |
| `icon`    | Icon to display in the alert                         |
| `default` | Alert content when not using title/description props |
