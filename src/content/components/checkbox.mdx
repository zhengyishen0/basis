## Overview

The Checkbox component provides a styled checkbox input with support for checked, unchecked, indeterminate, and disabled states. Features smooth animations, a visible checkmark indicator, and full Alpine.js integration for reactive behavior.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `id` | `string` | - | Unique identifier for the checkbox |
| `name` | `string` | `id` | Name attribute for form submission |
| `checked` | `boolean` | `false` | Whether the checkbox is checked |
| `indeterminate` | `boolean` | `false` | Whether the checkbox is in indeterminate state |
| `disabled` | `boolean` | `false` | Whether the checkbox is disabled |
| `label` | `string` | - | Label text displayed next to the checkbox |
| `class` | `string` | - | Additional CSS classes |

### Alpine.js Integration

The component supports all Alpine.js directives and automatically integrates with Alpine's reactivity system:

- Use `x-model` for two-way data binding
- Access `isChecked`, `isIndeterminate`, and `isDisabled` reactive properties
- Component automatically handles state management and updates

### Accessibility Features

- Proper ARIA attributes and roles
- Keyboard navigation support
- Screen reader compatible
- Focus management with visible focus indicators
- Semantic HTML with hidden native input for form compatibility

#### Advanced Alpine.js Usage

```astro
<div x-data="{
    preferences: {
        newsletter: false,
        notifications: true,
        analytics: false
    },
    toggleAll() {
        const allChecked = Object.values(this.preferences).every(v => v);
        Object.keys(this.preferences).forEach(key => {
            this.preferences[key] = !allChecked;
        });
    }
}">
    <button x-on:click="toggleAll()" class="mb-4 px-3 py-1 bg-primary text-primary-foreground rounded">
        Toggle All
    </button>
    
    <div class="space-y-3">
        <Checkbox label="Newsletter" x-model="preferences.newsletter" />
        <Checkbox label="Push Notifications" x-model="preferences.notifications" />
        <Checkbox label="Analytics" x-model="preferences.analytics" />
    </div>
</div>
```
