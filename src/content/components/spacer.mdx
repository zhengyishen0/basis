## Overview

The Spacer component is a flexible utility that grows to fill available space in flex layouts. It uses `flex-1 min-w-0` to expand and push elements apart, making it perfect for creating responsive layouts without manually calculating spacing.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `class` | `string` | - | Additional CSS classes |
| `...rest` | `any` | - | All other HTML attributes |

### Base Classes

- **`flex-1`**: Grows to fill available space in flex container
- **`min-w-0`**: Prevents flex item from having a minimum intrinsic width

## Design Principles

Spacer is a **simple utility component** that:
- Uses `flex-1` to grow and fill available space
- Uses `min-w-0` to prevent width calculation issues
- Works in both horizontal (Row) and vertical (Column) flex layouts
- Invisible by default - purely functional spacing
- Can be customized with additional classes if needed

## Common Use Cases

- **Navigation Bars**: Push menu items to opposite ends
- **Card Footers**: Align action buttons to the bottom
- **Modal Dialogs**: Separate cancel and confirm buttons  
- **List Items**: Push status indicators to the right
- **Product Cards**: Align prices and buttons consistently
- **Split Layouts**: Create flexible space between sections
- **Responsive Designs**: Maintain spacing without media queries

## Important Notes

- **No gap in parent**: When using Spacer, set `gap-0` on the parent flex container to avoid double spacing
- **Flex containers only**: Spacer only works within flex containers (Row, Column, or custom flex divs)
- **Multiple spacers**: Use multiple spacers to distribute space evenly between multiple elements
- **Minimum width**: The `min-w-0` prevents content overflow issues in tight spaces

## Comparison with CSS Alternatives

| Method | Use Case | Benefits |
|--------|----------|----------|
| `Spacer` | Dynamic layouts | No calculations, responsive, works with Alpine.js |
| `justify-between` | Fixed 2-item layouts | Simpler CSS, built into flex |
| `margin-left: auto` | Single push to right | Direct CSS approach |
| `gap` classes | Equal spacing | Uniform spacing between all items |
